# Audit Output Compliance

## MANDATORY Output Requirements

Every audit report MUST include:

### 1. Disclaimer (REQUIRED - First Section)
```
This report provides a risk assessment based on historical Federal Court 
jurisprudence. It does NOT predict outcomes or guarantee visa issuance. 
Officers retain discretion. We assess judicial defensibility only.
```

### 2. Defensibility Score
- Score: 0-100
- Clear rationale
- Confidence level

### 3. Evidence Checklist
Must include all three categories:
- **Baseline**: Existing documents
- **Live**: Obtainable documents
- **Strategic**: Recommended additions

### 4. Strategist Report
- Strengths
- Weaknesses
- Risk mitigation recommendations

### 5. Gatekeeper Review
- Compliance issues
- Refusal triggers
- Required fixes

## PROHIBITED Language
- "guaranteed success"
- "will definitely be approved"
- "100% chance"
- "assured outcome"
- Any promise of specific result

## Source Attribution
- All citations must be verifiable
- Note confidence level (High/Medium/Low)
- Include case citation format: "Smith v. Canada (MCI), 2023 FC 123"
