# 学习许可文档清单

> 基于 IMM 5483 (境外申请) 和 IMM 5555 (境内申请)

## 一、必需文件 (Required Documents)

### 1. 身份文件

| 文件 | 要求 | 备注 |
|------|------|------|
| 护照 | 有效期覆盖学习期+6个月 | 彩色扫描，所有有签证/印章页 |
| 护照照片 | 符合IRCC规格 (35x45mm) | 白底，6个月内 |
| 身份证 | 正反面复印件 | 中国申请人 |

### 2. 录取文件

| 文件 | 要求 | 备注 |
|------|------|------|
| 录取通知书 (LOA) | 来自DLI机构 | 必须包含DLI编号 |
| 省级认证信 (PAL) | 2024年1月22日后必需 | 有效期90-120天 |
| CAQ | 魁北克申请人 | 费用$117 |

**PAL豁免情况：**
- 硕士或博士项目
- K-12学生
- 交换生或访问学者
- 境内学签延期

### 3. 财务文件

| 文件 | 要求 | SDS特别要求 |
|------|------|-------------|
| 银行存款证明 | 金额覆盖学费+生活费 | - |
| 银行对账单 | 4-6个月历史 | - |
| GIC证明 | $20,635 | **必需** |
| 学费缴纳证明 | 第一年学费 | **必需** |
| 担保人收入证明 | 在职证明/纳税证明 | - |
| 资金来源说明 | 大额存入需解释 | - |

**2024年资金标准：**
- 生活费：$20,635/年
- 配偶：+$4,000/年
- 子女：+$3,000/年/人
- 魁北克：$24,000/年

### 4. 教育文件

| 文件 | 要求 | 备注 |
|------|------|------|
| 最高学历证书 | 毕业证+学位证 | 需认证翻译 |
| 成绩单 | 完整版 | 需认证翻译 |
| 语言成绩 | 2年内有效 | SDS要求IELTS 6.0+ |

### 5. 申请表格

| 表格 | 用途 |
|------|------|
| IMM 1294 | 学习许可申请表 |
| IMM 5645/5707 | 家庭信息表 |
| IMM 5257 | 临时居民签证(如需) |
| IMM 5476 | 代理人使用表(如适用) |
| IMM 5646 | 监护声明(未成年人) |

## 二、支持性文件 (Supporting Documents)

### 6. 学习计划

- [ ] 1-2页详细学习计划
- [ ] 包含：教育背景、选择理由、职业规划、回国计划

### 7. 工作相关 (如有)

- [ ] 在职证明
- [ ] 工作推荐信
- [ ] 请假批准信
- [ ] 工作合同

### 8. 约束力证明

- [ ] 房产证
- [ ] 车辆登记证
- [ ] 家庭成员信息
- [ ] 回国工作邀请(如有)

## 三、特殊情况文件

### 9. 未成年人 (18岁以下)

- [ ] IMM 5646 监护声明
- [ ] 加拿大监护人同意书 (公证)
- [ ] 父母同意书 (公证)
- [ ] 父母护照复印件
- [ ] 出生证明

### 10. 有拒签历史

- [ ] 之前的拒签信
- [ ] GCMS notes (如有)
- [ ] 解释信
- [ ] 新的支持材料

### 11. 配偶/子女随行

**配偶：**
- [ ] 配偶护照
- [ ] 结婚证
- [ ] 配偶工签申请(如适用)

**子女：**
- [ ] 子女护照
- [ ] 出生证明
- [ ] 疫苗记录

## 四、文件准备标准

### 翻译要求
- 所有非英语/法语文件需认证翻译
- 翻译件与原件一起提交
- 翻译人员资质证明

### 公证要求
- 父母同意书 (未成年人)
- 监护声明
- 某些财务文件

### 文件格式
- PDF格式优先
- 单个文件最大4MB
- 清晰可读
- 使用英文描述性文件名

## 五、提交前检查清单

### 完整性
- [ ] 所有必需文件齐全
- [ ] 文件在有效期内
- [ ] 翻译件完整准确

### 一致性
- [ ] 姓名拼写一致
- [ ] 日期信息一致
- [ ] 金额数字一致
- [ ] 项目信息匹配

### 常见错误
- ❌ 护照即将过期
- ❌ PAL已过期或即将过期
- ❌ 资金证明金额不足
- ❌ 缺少翻译件
- ❌ 照片不符规格
