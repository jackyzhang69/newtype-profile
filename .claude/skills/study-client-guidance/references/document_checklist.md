## 一、境外申请 (IMM 5483 + IMM 1294)

> **适用人群**: 在加拿大境外申请学签的申请人

### 1.1 必需表格

| 表格编号 | 名称 | 用途 |
|----------|------|------|
| **IMM 5483** | Document Checklist (Outside Canada) | 文件清单确认 |
| **IMM 1294** | Application for Study Permit (Outside) | 学签申请主表 |
| IMM 5645 | Family Information | 家庭信息表 |
| IMM 5257 | Application for TRV | 临时居民签证（如需） |
| IMM 5476 | Use of Representative | 代理人授权（如适用） |

### 1.2 身份文件

| 文件 | 要求 | 致命性 |
|------|------|--------|
| ✅ 护照 | 有效期覆盖学习期 + 6 个月 | Fatal |
| ✅ 护照照片 | IRCC 规格 (35x45mm, 白底, 6个月内) | Fatal |
| ✅ 身份证复印件 | 正反面彩色扫描 | 中国申请人必需 |

### 1.3 录取与省级文件

| 文件 | 要求 | 备注 |
|------|------|------|
| ✅ **录取通知书 (LOA)** | 来自 DLI 机构，包含 DLI 编号 | Fatal |
| ✅ **省级认证信 (PAL)** | 2024年1月22日后必需 | 见豁免情况 |
| ✅ CAQ | 魁北克省学校必需 | $117 费用 |

**PAL 豁免情况（境外申请）：**
- ✅ 硕士或博士学位项目
- ✅ K-12 学生（幼儿园至高中）
- ✅ 交换生或访问学者
- ❌ ~~境内延期~~ （仅适用于境内申请）

### 1.4 财务文件

| 文件 | 常规通道 | SDS 通道 |
|------|----------|----------|
| 银行存款证明 | ✅ 必需 | ✅ 必需 |
| 银行对账单 (4-6个月) | ✅ 必需 | ✅ 必需 |
| **GIC 证明** | ❌ 可选 | ✅ **必需** ($20,635) |
| **学费缴纳证明** | ❌ 可选 | ✅ **必需** (第一年全额) |
| 担保人收入证明 | ✅ 建议 | ✅ 建议 |
| 资金来源说明 | 大额存入时必需 | 大额存入时必需 |

**2024-2025 资金标准：**

| 项目 | 金额 (CAD) | 备注 |
|------|------------|------|
| 生活费（单身） | **$20,635/年** | 2024年1月1日起 |
| 配偶附加 | +$4,000/年 | |
| 子女附加 | +$3,000/年/人 | |
| **魁北克生活费** | **$24,000/年** | 高于其他省份 |

### 1.5 教育文件

| 文件 | 要求 | 备注 |
|------|------|------|
| ✅ 最高学历证书 | 毕业证 + 学位证 | 需认证翻译 |
| ✅ 成绩单 | 完整版 | 需认证翻译 |
| ✅ 语言成绩 | 2年内有效 | SDS 要求 IELTS 6.0+ |

## 二、境内申请 (IMM 5555 + IMM 5709)

> **适用人群**: 已在加拿大境内，申请学签延期或首次学签

### 2.1 必需表格

| 表格编号 | 名称 | 用途 |
|----------|------|------|
| **IMM 5555** | Document Checklist (Inside Canada) | 文件清单确认 |
| **IMM 5709** | Application to Change/Extend Stay as Student | 学签延期/变更主表 |
| IMM 5645 | Family Information | 家庭信息表 |
| IMM 5476 | Use of Representative | 代理人授权（如适用） |

### 2.2 ⭐ 境内申请特殊要求

| 文件 | 要求 | 说明 |
|------|------|------|
| ✅ **当前身份证明** | **境内申请必需** | 见下表 |
| ✅ PAL | **延期申请豁免** | 首次境内申请仍需 |

**当前身份证明类型：**

| 身份类型 | 证明文件 | 备注 |
|----------|----------|------|
| 学生 | 当前有效学签 | 延期申请 |
| 工签持有人 | 当前有效工签 | 转学签 |
| 访客 | Visitor Record | 访客转学签 |
| 隐含身份 | 原许可 + 延期申请回执 | 许可过期但已提交延期 |

**隐含身份 (Implied Status) 规则：**
- 原许可有效时提交了延期申请
- 原许可已过期但延期申请待审
- 可以继续在加拿大合法停留
- 需要保留延期申请确认

### 2.3 PAL 豁免规则（境内申请）

| 申请类型 | PAL 要求 |
|----------|----------|
| **学签延期** | ✅ **完全豁免** |
| **同校转专业** | ✅ 豁免 |
| **同省转学** | ⚠️ 需确认 |
| **跨省转学** | ❌ 可能需要 |
| **访客/工签首次转学签** | ❌ 需要 PAL |

## 三、SDS 快速通道特殊要求

> **适用国家**: 中国、印度、巴基斯坦、菲律宾、越南等 14 国
> 详见 `country_specific_requirements.md`

### 3.1 SDS vs 常规通道对比

| 对比项 | SDS | 常规通道 |
|--------|-----|----------|
| 处理时间 | ~20 个工作日 | 8-16 周 |
| GIC | **$20,635 必需** | 可选 |
| 学费 | **第一年全额必需** | 可选 |
| 语言 | **IELTS 6.0+ 必需** | 可选或较低 |
| 体检 | **申请前完成** | 申请后通知 |
| 适用人群 | 14 国居民 | 所有申请人 |

### 3.2 SDS 必需文件清单

| 文件 | 具体要求 | 证明文件 |
|------|----------|----------|
| ✅ **GIC** | $20,635 CAD | 认可银行 GIC 确认函 |
| ✅ **学费证明** | 第一年学费 100% 缴纳 | 学校收据或确认信 |
| ✅ **语言成绩** | IELTS Academic 6.0+ (所有单项) | 成绩单原件 |
| | 或 CELPIP 7+, PTE 60+, TOEFL 83+ | |
| ✅ **体检** | 申请前完成 | eMedical 回执 |

### 3.3 SDS 认可语言成绩

| 考试类型 | 最低要求 |
|----------|----------|
| IELTS Academic | 总分 6.0，每项 ≥ 6.0 |
| IELTS General Training | 每项 ≥ 6.0 |
| CELPIP General | 每项 ≥ 7 |
| PTE Academic | 总分 ≥ 60 |
| TOEFL iBT | 总分 ≥ 83 |
| TEF Canada | 每项 NCLC ≥ 7 |
| TCF Canada | 每项 NCLC ≥ 7 |

## 四、支持性文件

### 4.1 学习计划 (Study Plan)

**必须包含的内容：**
- [ ] 教育背景简述
- [ ] 为什么选择加拿大
- [ ] 为什么选择这所学校和专业
- [ ] 学习期间计划
- [ ] 毕业后职业规划
- [ ] 回国计划和理由

**避免的内容：**
- ❌ 提及移民意图
- ❌ 使用模板化语言
- ❌ 夸大或虚假信息

### 4.2 回国约束力证明

| 类型 | 文件 | 说明 |
|------|------|------|
| 财产 | 房产证 | 证明资产根基 |
| 财产 | 车辆登记证 | |
| 家庭 | 家庭成员信息 | 父母、配偶、子女 |
| 工作 | 回国工作邀请 | 如果有 |
| 工作 | 当前工作请假批准 | 证明保留职位 |

### 4.3 工作相关文件（如有）

- [ ] 在职证明（公司抬头信纸）
- [ ] 工作推荐信
- [ ] 请假批准信
- [ ] 工作合同
- [ ] 纳税证明

## 六、文件准备标准

### 6.1 翻译要求

| 要求 | 说明 |
|------|------|
| 所有非英语/法语文件 | 需认证翻译 |
| 翻译件格式 | 与原件一起提交 |
| 翻译人员 | 需提供资质证明 |
| 翻译声明 | 声明翻译准确完整 |

### 6.2 公证要求

必须公证的文件：
- 父母同意书（未成年人）
- 监护声明
- 某些财务文件（视情况）
- 亲属关系证明

### 6.3 文件格式

| 项目 | 要求 |
|------|------|
| 格式 | PDF 优先 |
| 大小 | 单个文件最大 4MB |
| 清晰度 | 所有文字清晰可读 |
| 文件名 | 英文描述性名称 |

## 七、提交前检查清单

### 7.1 完整性检查

- [ ] 所有必需文件齐全
- [ ] 表格填写完整无遗漏
- [ ] 签名处已签名
- [ ] 文件在有效期内
- [ ] 翻译件完整准确

### 7.2 一致性检查

- [ ] 姓名拼写一致（护照、LOA、银行文件）
- [ ] 出生日期一致
- [ ] 地址信息一致
- [ ] 项目信息匹配（LOA、PAL、学费收据）
- [ ] 金额数字一致（资金证明）

### 7.3 境内/境外检查

**境外申请：**
- [ ] IMM 5483 + IMM 1294 已填写
- [ ] PAL 有效（或符合豁免）
- [ ] 生物识别预约已安排

**境内申请：**
- [ ] IMM 5555 + IMM 5709 已填写
- [ ] 当前身份证明有效
- [ ] 如有隐含身份，保留申请确认

### 7.4 SDS 特别检查

- [ ] GIC 金额 ≥ $20,635
- [ ] 学费已全额缴纳
- [ ] 语言成绩符合要求
- [ ] 体检已完成

### 7.5 常见错误 ❌

| 错误 | 后果 |
|------|------|
| 护照即将过期 | 可能被拒 |
| PAL 过期或即将过期 | Fatal |
| 资金证明金额不足 | 高风险 |
| 缺少翻译件 | 文件不完整 |
| 照片不符规格 | 需要重新提交 |
| 表格填写不完整 | 申请被退回 |
| 姓名拼写不一致 | 需要解释或更正 |
