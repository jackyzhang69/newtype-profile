## 一、境外申请 (IMM 5483 + IMM 1294)

> **适用人群**: 在加拿大境外申请学签的申请人

### 1.1 必需表格

| 表格编号 | 名称 | 用途 |
|----------|------|------|
| **IMM 5483** | Document Checklist (Outside Canada) | 文件清单确认 |
| **IMM 1294** | Application for Study Permit (Outside) | 学签申请主表 |
| IMM 5645 | Family Information | 家庭信息表 |
| IMM 5257 | Application for TRV | 临时居民签证（如需） |
| IMM 5476 | Use of Representative | 代理人授权（如适用） |

### 1.2 身份文件

| 文件 | 要求 | 致命性 |
|------|------|--------|
| ✅ 护照 | 有效期覆盖学习期 + 6 个月 | Fatal |
| ✅ 护照照片 | IRCC 规格 (35x45mm, 白底, 6个月内) | Fatal |
| ✅ 身份证复印件 | 正反面彩色扫描 | 中国申请人必需 |

### 1.3 录取与省级文件

| 文件 | 要求 | 备注 |
|------|------|------|
| ✅ **录取通知书 (LOA)** | 来自 DLI 机构，包含 DLI 编号 | Fatal |
| ✅ **省级认证信 (PAL)** | 2024年1月22日后必需 | 见豁免情况 |
| ✅ CAQ | 魁北克省学校必需 | $117 费用 |

**PAL 豁免情况（境外申请）：**
- ✅ 硕士或博士学位项目
- ✅ K-12 学生（幼儿园至高中）
- ✅ 交换生或访问学者
- ❌ ~~境内延期~~ （仅适用于境内申请）

### 1.4 财务文件

> **注意**: SDS 已于 2024年11月8日关闭，现在只有常规通道

| 文件 | 要求 | 备注 |
|------|------|------|
| ✅ 银行存款证明 | 必需 | 金额覆盖学费+生活费 |
| ✅ 银行对账单 | 必需 (4-6个月) | 证明资金历史 |
| ⭐ GIC 证明 | **强烈建议** ($20,635) | 虽不再强制，但为强有力证明 |
| ⭐ 学费缴纳证明 | **强烈建议** | 第一年学费预付 |
| ✅ 担保人收入证明 | 建议 | 如使用担保 |
| ✅ 资金来源说明 | 大额存入时必需 | 解释资金来源 |

**2024-2025 资金标准：**

| 项目 | 金额 (CAD) | 备注 |
|------|------------|------|
| 生活费（单身） | **$20,635/年** | 2024年1月1日起 |
| 配偶附加 | +$4,000/年 | |
| 子女附加 | +$3,000/年/人 | |
| **魁北克生活费** | **$24,000/年** | 高于其他省份 |

### 1.5 教育文件

| 文件 | 要求 | 备注 |
|------|------|------|
| ✅ 最高学历证书 | 毕业证 + 学位证 | 需认证翻译 |
| ✅ 成绩单 | 完整版 | 需认证翻译 |
| ⭐ 语言成绩 | 2年内有效 | 虽不强制，但强烈建议有 |

## 二、境内申请 (IMM 5555 + IMM 5709)

> **适用人群**: 已在加拿大境内，申请学签延期或首次学签

### 2.1 必需表格

| 表格编号 | 名称 | 用途 |
|----------|------|------|
| **IMM 5555** | Document Checklist (Inside Canada) | 文件清单确认 |
| **IMM 5709** | Application to Change/Extend Stay as Student | 学签延期/变更主表 |
| IMM 5645 | Family Information | 家庭信息表 |
| IMM 5476 | Use of Representative | 代理人授权（如适用） |

### 2.2 ⭐ 境内申请特殊要求

| 文件 | 要求 | 说明 |
|------|------|------|
| ✅ **当前身份证明** | **境内申请必需** | 见下表 |
| ✅ PAL | **延期申请豁免** | 首次境内申请仍需 |

**当前身份证明类型：**

| 身份类型 | 证明文件 | 备注 |
|----------|----------|------|
| 学生 | 当前有效学签 | 延期申请 |
| 工签持有人 | 当前有效工签 | 转学签 |
| 访客 | Visitor Record | 访客转学签 |
| 隐含身份 | 原许可 + 延期申请回执 | 许可过期但已提交延期 |

**隐含身份 (Implied Status) 规则：**
- 原许可有效时提交了延期申请
- 原许可已过期但延期申请待审
- 可以继续在加拿大合法停留
- 需要保留延期申请确认

### 2.3 PAL 豁免规则（境内申请）

| 申请类型 | PAL 要求 |
|----------|----------|
| **学签延期** | ✅ **完全豁免** |
| **同校转专业** | ✅ 豁免 |
| **同省转学** | ⚠️ 需确认 |
| **跨省转学** | ❌ 可能需要 |
| **访客/工签首次转学签** | ❌ 需要 PAL |

## 三点五、Co-op 工作许可 (R186(w)) - 免费附加选项

> **⚠️ 很多学生不知道的"隐藏福利"**
>
> 如果项目包含 Co-op 或实习组件，可以在申请学签的**同时**申请 Co-op 工作许可。
> 这是一个"免费附加"选项，无需额外费用！

### 3.5.1 Co-op 工签资格

| 条件 | 要求 |
|------|------|
| ✅ 项目包含 Co-op | LOA 必须明确标注 Co-op/Work Placement |
| ✅ Co-op 为必修 | 必须是项目要求，非选修 |
| ✅ Co-op 时长 ≤ 50% | 工作学期不能超过项目总时长的一半 |

### 3.5.2 申请流程

```
申请学签时：
├── 在申请中勾选 "Co-op Work Permit" 选项
├── 上传学校出具的 Co-op 证明信
└── 无需额外费用
```

### 3.5.3 需要的额外文件

| 文件 | 来源 | 用途 |
|------|------|------|
| ✅ Co-op 证明信 | 学校国际学生办公室 | 证明项目包含 Co-op |
| ✅ 项目大纲 | 学校网站/招生材料 | 显示 Co-op 学期安排 |

### 3.5.4 常见错误

| 错误 | 后果 |
|------|------|
| ❌ 学签获批后才申请 Co-op 工签 | 需单独申请，费用 $155 |
| ❌ 项目不包含强制 Co-op | 申请被拒 |
| ❌ 未提供 Co-op 证明 | 可能被拒或延迟 |

## 四、支持性文件

- [ ] 教育背景简述

## 七、提交前检查清单

### 7.1 完整性检查

- [ ] 所有必需文件齐全
- [ ] 表格填写完整无遗漏
- [ ] 签名处已签名
- [ ] 文件在有效期内
- [ ] 翻译件完整准确

### 7.2 一致性检查

- [ ] 姓名拼写一致（护照、LOA、银行文件）
- [ ] 出生日期一致
- [ ] 地址信息一致
- [ ] 项目信息匹配（LOA、PAL、学费收据）
- [ ] 金额数字一致（资金证明）

### 7.3 境内/境外检查

**境外申请：**
- [ ] IMM 5483 + IMM 1294 已填写
- [ ] PAL 有效（或符合豁免）
- [ ] 生物识别预约已安排

**境内申请：**
- [ ] IMM 5555 + IMM 5709 已填写
- [ ] 当前身份证明有效
- [ ] 如有隐含身份，保留申请确认

### 7.4 ~~SDS 特别检查~~ — 强化材料建议

> **注意**: SDS 已于 2024年11月8日关闭，以下不再是强制要求，但仍为强有力的支持材料

- [ ] GIC 金额 ≥ $20,635 (建议)
- [ ] 学费已缴纳 (建议)
- [ ] 语言成绩 (强烈建议)
- [ ] 体检已完成 (可选)

### 7.5 常见错误 ❌

| 错误 | 后果 |
|------|------|
| 护照即将过期 | 可能被拒 |
| PAL 过期或即将过期 | Fatal |
| 资金证明金额不足 | 高风险 |
| 缺少翻译件 | 文件不完整 |
| 照片不符规格 | 需要重新提交 |
| 表格填写不完整 | 申请被退回 |
| 姓名拼写不一致 | 需要解释或更正 |

### 7.6 表格填写技巧 ✨

> **来自律师的实用建议**

| 技巧 | 说明 |
|------|------|
| ⭐ **预计完成日期 +90 天** | 比 LOA 结束日期多加 3 个月缓冲 |
| ⭐ **就业历史无空白** | 用 "Full-time Student" 填补学习期间 |
| ⭐ **Co-op 同时申请** | 如项目有 Co-op，与学签一起申请 |

### 7.7 境内申请关键提醒 ⚠️

> **CRITICAL: IP 地址检查**

| 要求 | 说明 |
|------|------|
| ✅ **必须从加拿大境内提交** | IRCC 系统检测 IP 地址 |
| ✅ 关闭 VPN 或使用加拿大服务器 | 确保 IP 显示加拿大 |
| ❌ **绝对不要**回国期间提交境内申请 | 可能导致自动拒绝 |
| ❌ **不要**让境外家人帮点提交 | 位置不匹配会被标记 |

**提交前检查**: 访问 whatismyip.com 确认 IP 显示加拿大位置
