# 学习许可获批后检查清单

> **最后更新**: 2026-01-26
> 
> **⚠️ 重要**: 学签获批后还有重要步骤需要完成！
> 很多学生以为获批就结束了，实际上遗漏以下步骤可能导致入境问题或后续麻烦。

---

## 一、立即检查: 学签 Remarks/Conditions (批注/条件)

> **⚠️ 这是最容易被忽略但最重要的一步！**

### 1.1 什么是 Remarks/Conditions?

学习许可上会有一个 "Remarks" 或 "Conditions" 部分，包含：
- 学习限制条件
- 工作许可限制
- 特殊要求
- 报到要求

### 1.2 必须检查的内容

| 检查项 | 说明 | 如果不符合 |
|--------|------|------------|
| **DLI 名称正确** | 学签上的学校必须与 LOA 匹配 | 联系 IRCC 更正 |
| **项目级别正确** | 学位/证书类型正确 | 联系 IRCC 更正 |
| **工作许可条款** | 是否允许校内/校外工作 | 遵守限制 |
| **条件码** | 如有特殊条件码，了解含义 | 查阅 IRCC 指南 |

### 1.3 常见条件码解读

| 条件码 | 含义 | 重要性 |
|--------|------|--------|
| **"May accept employment on or off campus..."** | 允许工作，每周不超过20小时（学期内） | 常见 |
| **"Must remain enrolled as a full-time student"** | 必须保持全日制 | 重要 |
| **"Limited to [specific institution]"** | 只能在指定学校学习 | 重要 |
| **"Co-op/Internship work permit"** | 包含 Co-op 工签 | 检查是否需要 |

### 1.4 如果发现错误

```
发现问题 → 不要使用此学签入境
          → 立即联系 IRCC Web Form
          → 说明错误并请求更正
          → 等待更正后的学签
```

## 二、临时居民签证 (TRV) 更新

### 2.1 谁需要 TRV?

| 国籍 | TRV 需求 | 说明 |
|------|----------|------|
| 中国 | ✅ 需要 | 必须有有效 TRV 才能入境 |
| 印度 | ✅ 需要 | |
| 美国 | ❌ 不需要 | 免签 |
| 英国 | ❌ 不需要 | 免签 |

### 2.2 TRV 检查清单

- [ ] **TRV 有效期**: 是否覆盖入境日期?
- [ ] **TRV 类型**: 是否为多次入境 (Multiple Entry)?
- [ ] **TRV 与学签匹配**: 如果学签是新的，TRV 可能也需要更新

### 2.3 TRV 更新时机

| 场景 | 是否需要更新 TRV | 建议 |
|------|------------------|------|
| 首次获得学签 | ✅ 是（通常随学签一起批） | 确认收到 |
| 学签延期（境内） | ⚠️ 可能 | 如果计划离开加拿大，需要 |
| 学签变更（境内） | ⚠️ 可能 | 如果 TRV 已过期且计划出境 |

### 2.4 何时申请新 TRV

```
境内延期学签获批后：
├── 计划在加拿大境内直到学签到期 → 不急于更新 TRV
├── 计划期间回国探亲 → 出境前更新 TRV
└── TRV 已过期 → 如需出境，必须先更新 TRV
```

## 三、社会保险号 (SIN) 更新

### 3.1 什么是 SIN?

- **Social Insurance Number**: 加拿大工作和报税必需
- **学生工作需要**: 校内外工作都需要有效 SIN
- **有效期限制**: 临时居民的 SIN 有效期与身份相关

### 3.2 何时需要更新 SIN

| 场景 | 是否需要更新 | 说明 |
|------|--------------|------|
| 学签延期获批 | ✅ 是 | SIN 有效期与学签挂钩 |
| 首次获得学签 | ✅ 需要申请 | 到加拿大后申请 |
| 学签条件变更 | ✅ 可能 | 如工作许可条款变化 |

### 3.3 SIN 更新流程

```
1. 学签延期/更新获批
         ↓
2. 收到新的学签（电子版或纸质版）
         ↓
3. 前往 Service Canada 办公室
   - 带上新学签
   - 带上护照
   - 带上旧 SIN 卡/确认信
         ↓
4. 获得更新的 SIN 确认
         ↓
5. 通知雇主（如有）新的 SIN 有效期
```

### 3.4 SIN 更新检查清单

- [ ] 新学签已收到
- [ ] 预约/前往 Service Canada
- [ ] 携带所有必需文件
- [ ] 获得新的 SIN 确认
- [ ] 通知雇主（如适用）

## 四、学校报到和注册

### 4.1 必须完成的步骤

| 步骤 | 时间要求 | 重要性 |
|------|----------|--------|
| **在 IRCC Portal 报到** | 入境后 | 可能需要 |
| **向学校报到** | 开学前 | 必需 |
| **提交新学签副本** | 尽快 | 重要 |
| **确认注册状态** | 开学前 | 必需 |

### 4.2 IRCC Portal 更新

- 如果通过在线申请获批，检查 IRCC 账户是否需要确认入境
- 部分情况需要在入境后更新状态

### 4.3 学校国际学生办公室

**必须向学校提交**:
- 新学签副本
- 护照身份页副本
- 入境章页面副本（如适用）

## 五、保险和健康覆盖

### 5.1 健康保险检查

| 省份 | 公共健康保险 | 等待期 | 建议 |
|------|--------------|--------|------|
| BC | MSP | 3个月 | 购买临时保险 |
| Ontario | OHIP | 不覆盖学生 | 必须购买私人保险 |
| Quebec | RAMQ | 覆盖法国学生 | 其他需私人保险 |
| Alberta | AHCIP | 通常覆盖 | 确认资格 |

### 5.2 保险检查清单

- [ ] 确认省级健康保险资格
- [ ] 如有等待期，购买临时保险
- [ ] 确认学校强制保险要求
- [ ] 保留保险证明

## 六、入境准备

### 6.1 入境必带文件

| 文件 | 必须 | 说明 |
|------|------|------|
| ✅ 护照 | 是 | 有效期足够 |
| ✅ 学签批准信/确认 | 是 | POE Letter 或打印件 |
| ✅ LOA 原件/副本 | 是 | 录取通知书 |
| ⭐ 资金证明 | 建议 | 可能被要求出示 |
| ⭐ 学习计划 | 建议 | 可能被询问 |
| ⭐ 住宿证明 | 建议 | 地址信息 |

### 6.2 入境口岸检查预期

入境官员可能询问：
- 学习什么项目?
- 在哪所学校?
- 资金来源?
- 住在哪里?
- 毕业后计划?

**提示**: 诚实、简洁、自信地回答。

## 七、完整检查清单汇总

### 获批后立即 (1-3 天内)

- [ ] 仔细阅读学签上的所有 Remarks/Conditions
- [ ] 确认学校名称和项目正确
- [ ] 确认工作许可条款
- [ ] 如有错误，立即联系 IRCC

### 出发前 (1-2 周内)

- [ ] 确认 TRV 有效（如适用）
- [ ] 打印或保存所有批准文件
- [ ] 准备入境所需所有文件
- [ ] 确认住宿和接机安排
- [ ] 购买机票（如尚未购买）

### 到达后 (前 30 天)

- [ ] 完成入境手续
- [ ] 向学校国际学生办公室报到
- [ ] 申请/更新 SIN
- [ ] 确认健康保险覆盖
- [ ] 开设银行账户（如需要）
- [ ] 获取手机号

### 开学前

- [ ] 确认课程注册
- [ ] 了解全日制要求
- [ ] 了解工作限制 (20小时/周)
- [ ] 保存所有重要文件副本

## 八、常见问题

### Q: 学签获批了但 TRV 没有，怎么办?

**A**: 如果你是需要 TRV 的国籍（如中国），通常学签和 TRV 会一起批准。如果只收到学签批准：
1. 检查邮件是否有 TRV 批准信
2. 检查护照是否已有有效 TRV（多次入境）
3. 如确实没有，需单独申请 TRV

### Q: 学签上的条件与我申请的不一样怎么办?

**A**: 
1. 不要使用此学签入境
2. 通过 IRCC Web Form 联系移民局
3. 说明错误并请求更正
4. 等待更正后再行动

### Q: 我在境内延期，需要更新 SIN 吗?

**A**: 是的，如果学签延期获批，应该更新 SIN 有效期。前往 Service Canada 并携带新学签。

### Q: 入境时学签不是贴在护照上吗?

**A**: 2024年后，学签以电子形式签发 (e-Study Permit)。入境时，边境官员会在系统中确认并在护照上盖章。建议打印批准信随身携带。
