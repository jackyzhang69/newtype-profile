# 担保人资金宣誓书指南 (Sponsorship Affidavit Guide)

> **最后更新**: 2026-01-26
> 
> **适用场景**: 当父母或其他担保人提供超过 50% 的学习资金时，**强烈建议**提供正式的资金担保宣誓书。

---

## 一、何时需要担保宣誓书

### 1.1 强制/强烈建议的情况

| 情况 | 需求程度 | 说明 |
|------|----------|------|
| 父母资助 > 50% 学费+生活费 | **强烈建议** | 签证官需要确认担保意愿 |
| 申请人无个人收入 | **强烈建议** | 需要证明资金来源和支持 |
| 资金在担保人名下 | **必需** | 证明资金可用于学生 |
| 担保人与申请人不同姓 | **必需** | 需要证明关系和支持意愿 |
| 多个担保人共同资助 | **强烈建议** | 明确各方责任 |

### 1.2 可选的情况

| 情况 | 需求程度 | 说明 |
|------|----------|------|
| 申请人有充足个人存款 | 可选 | 自己资金足够 |
| 获得全额奖学金 | 可选 | 奖学金覆盖全部费用 |
| 政府资助项目 | 可选 | 官方资助证明即可 |

---

## 二、宣誓书必备内容

### 2.1 基本要素

| 要素 | 必需 | 说明 |
|------|------|------|
| ✅ 担保人全名 | 是 | 与护照/身份证一致 |
| ✅ 与申请人关系 | 是 | 父/母/祖父母/其他亲属 |
| ✅ 担保金额 | 是 | 具体金额和覆盖范围 |
| ✅ 担保期限 | 是 | 覆盖整个学习期间 |
| ✅ 资金来源说明 | 是 | 工资/经营/投资等 |
| ✅ 担保人签名 | 是 | 手写签名 |
| ✅ 日期 | 是 | 签署日期 |
| ⭐ **流动性声明** | **强烈建议** | 见下文详解 |
| ⭐ 公证 | 建议 | 增加可信度 |

### 2.2 ⭐ 流动性声明 (Liquidity Clause) - 关键！

> **这是很多申请人忽略的关键点！**

**为什么需要流动性声明？**

签证官不仅看资金总额，还关心：
1. 资金是否**可以动用**（流动性）
2. 担保人是否**愿意动用**
3. 动用资金后是否**影响担保人生活**

**流动性声明应包含：**

```
我确认：
1. 上述资金为流动资产，可随时用于支付 [学生姓名] 的学费和生活费
2. 这些资金的使用不会影响我和家庭的正常生活水平
3. 我有能力在需要时将资金转移至加拿大银行账户
4. 我了解可能需要遵守外汇管理规定（如适用）
```

## 三、宣誓书模板

### 3.1 英文模板 (推荐)

```
FINANCIAL SPONSORSHIP AFFIDAVIT

I, [担保人全名], holder of [护照/身份证号码], residing at [详细地址], hereby declare and affirm as follows:

1. RELATIONSHIP
I am the [father/mother/grandfather/grandmother/other] of [申请人全名], born on [出生日期], holder of passport number [护照号].

2. FINANCIAL COMMITMENT
I hereby undertake to financially support [申请人姓名] for the entire duration of their studies in Canada, specifically for the [项目名称] program at [学校名称], from [开始日期] to [预计结束日期].

3. FUNDING AMOUNT
I commit to providing the following financial support:
- Tuition fees: CAD $[金额] per year
- Living expenses: CAD $[金额] per year
- Total commitment: CAD $[总金额] for [年数] years of study

4. SOURCE OF FUNDS
The funds for this sponsorship come from my:
- Employment income as [职位] at [公司名称]
- Business income from [公司/生意名称]
- Savings accumulated over [年数] years
- Investment returns from [投资类型]
[根据实际情况选择]

5. LIQUIDITY STATEMENT ⭐
I confirm that:
(a) The above-mentioned funds are liquid assets readily available for [申请人姓名]'s educational expenses;
(b) The use of these funds will not adversely affect my family's standard of living or financial stability;
(c) I am able to transfer funds to Canada as needed, in compliance with applicable foreign exchange regulations;
(d) I have attached bank statements and/or certificates demonstrating the availability of these funds.

6. DECLARATION
I make this affidavit conscientiously believing it to be true and knowing that it is of the same force and effect as if made under oath.

Signature: _______________________
Name: [担保人全名]
Date: [日期]
Place: [地点]

[公证处盖章/公证员签字 - 如适用]
```

### 3.2 中文模板 (需配合翻译件)

```
资金担保声明书

本人 [担保人全名]，身份证号码 [号码]，现居住于 [详细地址]，特此声明如下：

一、与申请人关系
本人系 [申请人全名]（出生日期：[日期]，护照号码：[号码]）之 [父亲/母亲/祖父/祖母/其他]。

二、资金担保承诺
本人承诺为 [申请人姓名] 在加拿大 [学校名称] 攻读 [项目名称] 期间（[开始日期] 至 [预计结束日期]）提供全部所需资金支持。

三、担保金额
本人承诺提供以下资金支持：
- 学费：每年 CAD $[金额]
- 生活费：每年 CAD $[金额]
- 总计：[年数] 年学习期间共 CAD $[总金额]

四、资金来源
上述资金来源于：
- 本人在 [公司名称] 担任 [职位] 的工资收入
- 本人经营 [公司/生意名称] 的营业收入
- 本人 [年数] 年来的积蓄
- 投资收益
[根据实际情况选择]

五、流动性声明 ⭐
本人确认：
(1) 上述资金为流动资产，可随时用于支付 [申请人姓名] 的学费和生活费；
(2) 使用这些资金不会影响本人及家庭的正常生活水平和财务稳定；
(3) 本人有能力按照外汇管理规定将资金转移至加拿大；
(4) 随附的银行证明文件可证明上述资金的可用性。

六、声明
本人谨此声明，以上所述均为真实情况，并愿承担相应法律责任。

签名：_______________________
姓名：[担保人全名]
日期：[日期]
地点：[地点]

[公证处盖章 - 如适用]
```

## 四、中国申请人特别注意事项

### 4.1 外汇管制相关

> **⚠️ 中国外汇管制对学签申请的影响**

| 规定 | 内容 | 对申请的影响 |
|------|------|--------------|
| 个人年度购汇额度 | 等值 5 万美元/人/年 | 多年学费可能需要多人或多年购汇 |
| 境外汇款用途 | 需要真实用途证明 | 需要录取通知书等 |
| 大额资金来源 | 可能需要解释 | 准备收入证明 |

### 4.2 流动性声明的中国特色补充

对于中国担保人，建议在流动性声明中增加：

```
关于资金转移至加拿大的补充说明：

本人了解中国外汇管理相关规定，并确认：
1. 本人已预留足够的购汇额度用于支付 [申请人姓名] 的学费和生活费
2. 必要时，本人可协调其他直系亲属的购汇额度以确保资金及时转移
3. 本人已了解通过银行汇款至加拿大的流程，并能在 [申请人姓名] 需要时及时办理

附：本人及 [配偶姓名/其他亲属姓名] 的身份证复印件，证明可用购汇额度
```

### 4.3 建议的配套文件

| 文件 | 用途 | 必需程度 |
|------|------|----------|
| 担保人身份证复印件 | 身份证明 | 必需 |
| 担保人护照复印件 | 身份证明（如有） | 建议 |
| 担保人收入证明 | 证明资金来源 | 必需 |
| 担保人银行流水 | 证明收入真实性 | 强烈建议 |
| 担保人纳税证明 | 辅助证明收入 | 建议 |
| 关系证明（户口本） | 证明与申请人关系 | 必需 |
| 公证书 | 增加可信度 | 建议 |

## 五、常见问题

### Q1: 一定要公证吗？

**A**: 不是强制要求，但**强烈建议公证**，特别是：
- 担保金额较大时
- 担保人与申请人不同姓时
- 使用中文版本时（公证翻译）

### Q2: 父母都要出宣誓书吗？

**A**: 取决于资金情况：
- 资金在一方名下 → 该方出具即可
- 资金在双方名下 → 建议双方分别出具或联合出具
- 双方共同担保 → 建议联合出具，明确各自承担比例

### Q3: 担保人不是父母可以吗？

**A**: 可以，但需要：
- 解释担保关系和原因
- 更详细的资金来源证明
- 可能需要额外的关系证明

### Q4: 宣誓书有效期是多久？

**A**: 建议：
- 签署日期距申请提交不超过 3 个月
- 如果申请延迟，可能需要更新

---

## 六、审计检查点

审计时应检查：

- [ ] 申请人是否需要担保人资助？
- [ ] 担保人资助比例是否超过 50%？
- [ ] 是否提供了担保宣誓书？
- [ ] 宣誓书是否包含流动性声明？
- [ ] 是否考虑了外汇管制因素（中国申请人）？
- [ ] 配套文件是否齐全？
