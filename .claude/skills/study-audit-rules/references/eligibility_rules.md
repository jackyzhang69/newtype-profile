# 学习许可资格检查规则

> **重要澄清**: "SPS (Student Partners Stream)" 不存在。该名称可能来自两个混淆：
> - **SPP (Student Partners Program)**: 已于 2018 年终止，被 SDS 取代
> - **SDS (Student Direct Stream)**: 当前有效的快速通道（见第三节）
> 
> 如果听到 "SPS"，请确认申请人实际指的是 SDS 还是常规学签通道。

## 一、基本资格要求 (IRPR 216)

### 1.1 申请人资格

| 条件 | 要求 | 致命性 |
|------|------|--------|
| 录取通知 | 被 DLI 机构正式录取 | Fatal |
| 学习期限 | 项目超过6个月 | Fatal |
| 财务能力 | 证明有足够资金 | Fatal |
| 临时意图 | 使移民官相信将在授权期满后离开 | Fatal |
| 真实学生 | 满足 R220 真实学生测试 | Fatal |

### 1.2 PAL 要求 (2024年1月22日起)

**省级认证信 (Provincial Attestation Letter) 要求：**

| 省份 | PAL 要求 | 处理时间 | 费用 |
|------|----------|----------|------|
| Ontario | 必需 | 2-4周 | 免费 |
| BC | 必需（学校申请） | 2-3周 | 免费 |
| Quebec | CAQ 替代 | 4-6周 | $117 |
| Alberta | 必需 | 1-2周 | 免费 |
| 其他省份 | 必需 | 1-3周 | 免费 |

**PAL 豁免情况 (完整列表)：**

| 豁免类别 | 具体条件 | 备注 |
|----------|----------|------|
| 硕士/博士 | Master's 或 PhD 项目 | 仅限学位项目，非研究生证书 |
| K-12 学生 | 幼儿园至12年级 | 包括小学、初中、高中 |
| 交换生 | 学校间交换协议 | 需要交换协议证明 |
| 访问学者 | 研究/访问目的 | 非学位项目 |
| **境内延期** | **已持有有效学签** | **境内申请延期全部豁免 PAL** |
| **境内转学** | **已持有有效学签** | **同省内转学豁免；跨省需确认** |

**⚠️ 关键区分：**
- **境外申请 (IMM 5483)**: PAL **必需**（除非上述豁免）
- **境内延期 (IMM 5555)**: PAL **完全豁免**

## 二、真实学生测试 (Genuine Student Test - R220)

### 2.1 法律基础

**IRPR 220 条款：**
> 签证官不得向外国人签发学习许可，除非确信该外国人：
> (a) 将在授权逗留期限结束时离开加拿大；
> (b) 是真正的学生，会积极追求其课程或学习项目。

### 2.2 评估维度

```
┌─────────────────────────────────────────────────────────┐
│                   真实学生评估框架                       │
├─────────────────────────────────────────────────────────┤
│  1. 学习计划合理性                                      │
│     - 与过往教育的逻辑连续性                            │
│     - 职业目标的可信度                                  │
│     - 选择加拿大/学校的合理理由                         │
│                                                         │
│  2. 财务能力真实性                                      │
│     - 资金来源合法性                                    │
│     - 担保人关系及支持意愿                              │
│     - 投资回报预期合理性                                │
│                                                         │
│  3. 与原籍国联系 (Ties to Home)                         │
│     - 家庭关系强度                                      │
│     - 财产和资产                                        │
│     - 就业前景                                          │
│                                                         │
│  4. 移民历史                                            │
│     - 之前的签证申请                                    │
│     - 旅行合规记录                                      │
│     - 在其他国家的身份状态                              │
└─────────────────────────────────────────────────────────┘
```

### 2.3 双重意图 (Dual Intent - IRPA 22(2))

**法律规定：**
> IRPA 22(2): 外国人寻求临时居民身份的意图与同时寻求成为永久居民的意图并不矛盾。

**关键要点：**
- 双重意图是被允许的
- 但仍需证明：如果永居申请失败，将离开加拿大
- 表述方式极其重要 (参见 Luk v Canada, 2024 FC 623)

## 三、财务证明要求

### 3.1 2024年资金标准

| 项目 | 金额 (CAD) | 备注 |
|------|------------|------|
| 生活费（单身） | $20,635/年 | 2024年1月起调整 |
| 配偶附加 | +$4,000/年 | |
| 每个子女 | +$3,000/年 | |
| 学费 | 实际金额 | 按录取通知书 |
| 返程机票 | ~$2,000 | 估算 |
| 魁北克生活费 | $24,000/年 | 高于其他省份 |

### 3.2 SDS (Student Direct Stream) 详细要求

> **历史背景**: SDS 于 2018 年取代了 SPP (Student Partners Program)。
> SPP 仅限特定合作院校；SDS 对所有 DLI 机构开放。
>
> ⚠️ **状态验证提醒**: 
> 在为申请人提供建议前，请通过官网确认 SDS 当前状态：
> https://www.canada.ca/en/immigration-refugees-citizenship/services/study-canada/study-permit/student-direct-stream.html
> IRCC 政策可能随时调整，本文档记录的是 2024-2025 年规则。

**SDS 适用国家 (14国完整列表)：**

| 地区 | 国家 |
|------|------|
| 亚洲 | **中国**、**印度**、**巴基斯坦**、**菲律宾**、**越南** |
| 美洲 | 巴西、哥伦比亚、哥斯达黎加、秘鲁、特立尼达和多巴哥、安提瓜和巴布达、圣文森特和格林纳丁斯 |
| 非洲 | 摩洛哥、塞内加尔 |

**SDS 资格要求：**

| 要求 | 详情 | 证明文件 |
|------|------|----------|
| GIC | **$20,635 CAD** | 参与银行出具的 GIC 证明 |
| 学费 | **第一年学费全额缴纳** | 学校收据或学费缴纳证明 |
| 语言 | **IELTS Academic 6.0+**（所有单项） | 或同等：CELPIP 7+, PTE 60+, TEF/TCF NCLC 7+ |
| 体检 | **提前完成** | 体检回执，申请前完成 |
| 无犯罪 | 部分国家要求提前提供 | 见 Visa Office Instructions |

**SDS vs 常规通道对比：**

| 对比项 | SDS | 常规通道 |
|--------|-----|----------|
| 处理时间 | ~20 个工作日 | 8-16 周 |
| 财务证明 | GIC + 学费（固定要求） | 灵活（银行存款等） |
| 语言要求 | IELTS 6.0+ 必须 | 可无或较低 |
| 体检时间 | 申请前 | 申请后通知 |
| 适用人群 | 14 国居民 | 所有申请人 |

**⚠️ 注意**: 如果不满足 SDS 要求，14 国申请人仍可通过常规通道申请。

### 3.3 资金来源证明

**可接受的资金来源：**
1. 个人储蓄（6个月历史最佳）
2. 父母/家庭担保
3. 奖学金/助学金
4. 政府资助
5. 银行贷款（需还款能力证明）

**资金历史要求：**
- 理想: 6个月以上
- 最低: 3-4个月
- 大额存入: 必须解释来源

## 四、风险评估框架

### 4.1 致命红旗 (Fatal Flags)

- [ ] 无有效 DLI 录取通知
- [ ] PAL 缺失或过期（需要时）
- [ ] 资金明显不足（< 50% 要求）
- [ ] 发现虚假文件
- [ ] 被发现有虚假陈述历史
- [ ] 不可入境（犯罪/安全/健康）

### 4.2 高风险红旗 (High Risk)

- [ ] 学习计划与背景严重不符（倒读）
- [ ] 年龄 > 35岁申请本科或以下
- [ ] 有多次学签拒签历史
- [ ] 家庭可整体移居（无回国约束）
- [ ] 之前在加拿大有违规记录
- [ ] 语言能力与项目要求严重不匹配
- [ ] 申请敏感专业无合理解释

### 4.3 中等风险红旗 (Medium Risk)

- [ ] 年龄 28-35 岁
- [ ] 有1-2次拒签历史
- [ ] 资金历史 < 3个月
- [ ] 学习计划描述模糊
- [ ] 无工作经验直接申请研究生
- [ ] 职业规划不明确
- [ ] 与原籍国联系证据不足

### 4.4 低风险指标 (Positive Factors)

- [+] 年轻单身（18-25岁）
- [+] 专业匹配且有逻辑进阶
- [+] 强约束力（家庭/财产/工作）
- [+] 良好旅行历史（美、英、申根等）
- [+] 知名大学/排名靠前项目
- [+] 奖学金或资助
- [+] 相关工作经验

## 五、拒签原因分析

### 5.1 2024-2025年 TOP 拒签原因

| 排名 | 原因 | 占比 | 对应代码 |
|------|------|------|----------|
| 1 | 学习计划不合理 | 35% | R220(b) |
| 2 | 财务能力不足 | 25% | R220 + funds |
| 3 | 不会离开加拿大 | 20% | R220(a) |
| 4 | 材料真实性存疑 | 10% | R40 |
| 5 | 语言能力不足 | 5% | - |
| 6 | 其他 | 5% | - |

### 5.2 常见拒签理由模板

**Purpose of Visit (访问目的):**
> "I am not satisfied that you will leave Canada at the end of your stay as a temporary resident."

**Study Plan (学习计划):**
> "I am not satisfied that you are a genuine student based on the study plan provided."

**Financial Resources (财务资源):**
> "I am not satisfied that you have sufficient funds to support yourself and any family members during your intended stay in Canada."

**Ties to Home Country (约束力):**
> "I am not satisfied based on your personal assets and financial status, employment prospects in your country of residence, and family ties and obligations."

## 七、检查流程

1. **资格筛查**: Fatal 条件检查 → 任何不满足即停止
2. **风险评估**: 红旗检查 → 记录所有风险点
3. **材料审核**: 文件完整性 → 标记缺失项
4. **综合评估**: 正面因素 vs 风险因素 → 生成风险评分
5. **建议生成**: 基于评估结果提供策略建议
