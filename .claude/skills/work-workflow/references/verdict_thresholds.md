# Work Permit Verdict Thresholds

## Verdict Mapping

| Score Range | Risk Level | Verdict | Action |
|-------------|------------|---------|--------|
| 90-100 | Excellent | **GO** | Proceed with confidence |
| 75-89 | Good | **APPROVE** | Proceed (Pro+ tier) |
| 60-74 | Adequate | **CAUTION** | Proceed with mitigations |
| 40-59 | Weak | **CAUTION** or **NO-GO** | Conditional or reject |
| 0-39 | Poor | **NO-GO** | Do not proceed as-is |

## Verdict Definitions

### GO (90-100)
- Valid LMIA or LMIA-exempt category
- Employer legitimacy verified
- Strong job offer credentials match
- No admissibility concerns
- **Action**: Submit application with standard preparation

### APPROVE (75-89) - Pro+ Tier
- LMIA/exempt status confirmed
- Minor documentation gaps addressable
- Credentials verified
- No fatal issues
- **Action**: Submit with noted mitigations

### CAUTION (60-74)
- LMIA/exempt status unclear
- Some employer legitimacy concerns
- Credential gaps need addressing
- Dual intent indicators present
- **Action**: Address specific gaps before submission

### CAUTION/NO-GO Boundary (40-59)
- LMIA validity questionable
- Employer legitimacy red flags
- Significant credential mismatch
- Strong immigration intent concerns
- **Decision Criteria**: Upgrade to CAUTION if LMIA and employer issues resolvable

### NO-GO (0-39)
- Invalid or expired LMIA
- Fraudulent employer indicators
- Credentials don't match job requirements
- Previous work permit violations
- **Action**: Do not submit; fundamental issues present

## Poison Pill Rules

These conditions automatically trigger NO-GO regardless of score:

1. **Invalid LMIA** - Expired, revoked, or fraudulent LMIA
2. **Employer Compliance Issues** - Employer on non-compliance list
3. **R200 Ineligibility** - Does not meet work permit requirements
4. **Active Removal Order** - Under removal proceedings
5. **Misrepresentation Finding** - Previous A40 finding
6. **Criminal Inadmissibility** - Serious criminality without rehabilitation

## Category-Specific Thresholds

### LMIA-Based Work Permits

| Factor | Score Impact |
|--------|--------------|
| LMIA positive, valid | Baseline |
| LMIA expiring within 60 days | -10 |
| High-wage position | +5 |
| Low-wage position | 0 |
| Seasonal agricultural | -5 (additional scrutiny) |

### PGWP (Post-Graduation Work Permit)

| Factor | Score Impact |
|--------|--------------|
| DLI program completed | Required (Poison Pill if not) |
| Applied within 180 days | Required |
| Full-time study maintained | Required |
| Co-op/internship component | +5 |

### ICT (Intra-Company Transfer)

| Factor | Score Impact |
|--------|--------------|
| 1 year continuous employment | Required |
| Specialized knowledge demonstrated | +10 |
| Executive/manager level | +10 |
| L-1 blanket equivalent | +5 |

### Open Work Permits (Spousal)

| Factor | Score Impact |
|--------|--------------|
| Principal valid status | Required |
| Genuine relationship | Per spousal scoring |
| Principal occupation eligible | Required |

## Score Override Conditions

| Condition | Override |
|-----------|----------|
| LMIA valid + Employer verified | Minimum score: 60 |
| Any dimension < 40 | Maximum score: 65 |
| Any Poison Pill present | Force NO-GO |
| All Poison Pills cleared | Allow score-based verdict |
