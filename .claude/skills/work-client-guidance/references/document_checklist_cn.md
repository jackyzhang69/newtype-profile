# 工作许可文件清单指南

**适用范围**: LMIA工作许可、PGWP、ICT、配偶开放工签、PNP工签
**语言**: 中文
**版本**: 1.0.0
**最后更新**: 2026年1月

---

## 一、通用文件要求

### 1.1 所有工作许可申请必需文件

| 文件类型 | 说明 | 格式要求 |
|----------|------|----------|
| 护照 | 有效期应超过工签期限 | 彩色扫描件，所有页面 |
| 照片 | 符合IRCC规格 | 35mm x 45mm，白色背景 |
| 申请表 | IMM 1295 (境外) 或 IMM 5710 (境内) | 电子填写，签名 |
| 家庭信息表 | IMM 5707 | 电子填写 |
| 代理人授权 | IMM 5476 (如使用代理) | 签名 |

### 1.2 身份和状态文件

| 文件类型 | 适用情况 | 说明 |
|----------|----------|------|
| 当前签证/许可 | 境内申请 | 有效的学签/工签/访客签证 |
| 入境章 | 境内申请 | 护照上的入境记录 |
| 出生证明 | 所有申请 | 公证翻译件 |
| 结婚证 | 如已婚 | 公证翻译件 |

## 二、LMIA工作许可文件清单

### 2.1 必需文件

| 文件 | 来源 | 说明 |
|------|------|------|
| LMIA批准信 | 雇主提供 | 包含LMIA编号 |
| 雇主Offer Letter | 雇主提供 | 包含职位、工资、工作地点、开始日期 |
| 雇佣合同 | 雇主提供 | 详细条款和条件 |

### 2.2 资格证明文件

| 文件类型 | 说明 | 重要性 |
|----------|------|--------|
| 学历证明 | 学位证书、毕业证书 | 必需 |
| 学历认证 | WES或其他认证机构 | 强烈建议 |
| 专业证书 | 与职位相关的执照/证书 | 如适用 |
| 工作经验证明 | 推荐信、工资单、税单 | 必需 |

### 2.3 工作经验证明要求

**推荐信应包含：**
- 公司抬头纸
- 雇主联系方式
- 雇佣日期（起止）
- 职位名称
- 工作职责描述
- 工资信息
- 全职/兼职状态
- 签名和日期

**示例格式：**
```
[公司抬头]

日期：[日期]

致相关方：

兹证明[申请人姓名]于[起始日期]至[结束日期]在本公司担任
[职位]。

工作职责包括：
- [职责1]
- [职责2]
- [职责3]

[申请人姓名]的年薪为[金额]，为全职员工。

如有任何问题，请联系[联系人姓名]，电话[电话号码]。

[签名]
[姓名]
[职位]
```

## 三、PGWP毕业工签文件清单

### 3.1 必需文件

| 文件 | 来源 | 说明 |
|------|------|------|
| 学校完成信 | 学校注册处 | 确认完成学业 |
| 正式成绩单 | 学校注册处 | 或学校网站截图 |
| 学习许可副本 | 申请人 | 当前或最近的学习许可 |

### 3.2 学校完成信要求

**完成信应包含：**
- 学校官方抬头
- 学生姓名和学号
- 项目名称和学位类型
- 项目开始日期
- 项目完成日期
- 项目总时长
- 全日制学习确认
- 学校DLI编号

### 3.3 特殊情况额外文件

| 情况 | 额外文件 |
|------|----------|
| 学习中断 | 医疗证明、学校确认信 |
| 项目变更 | 两个项目的确认信 |
| 学习许可过期 | 身份恢复费用收据 |
| 护照即将过期 | 新护照申请证明 |

### 3.4 PGWP文件核对清单

- [ ] 学校完成信
- [ ] 正式成绩单
- [ ] 学习许可副本
- [ ] 护照复印件（有效期页、签名页、入境章）
- [ ] 照片
- [ ] 申请表
- [ ] 申请费支付证明
- [ ] 开放工签费支付证明
- [ ] 身份恢复费收据（如适用）

### 3.5 PGWP关键时间验证

| 时间点 | 文件证明 |
|--------|----------|
| 最终成绩发布日期 | 成绩单日期 |
| 180天申请窗口 | 申请提交日期 |
| 学习许可有效期 | 学习许可副本 |

## 四、ICT公司内部调动文件清单

### 4.1 公司关系证明文件

| 文件 | 说明 | 重要性 |
|------|------|--------|
| 母公司注册文件 | 公司注册证、营业执照 | 必需 |
| 加拿大公司注册文件 | 公司注册证 | 必需 |
| 股权结构证明 | 股东名册、股权证书 | 必需 |
| 公司关系说明信 | 解释母子公司关系 | 必需 |

### 4.2 申请人资格证明

| 文件 | 说明 | 重要性 |
|------|------|--------|
| 雇佣证明信 | 母公司出具 | 必需 |

| 职位描述 | 当前和拟任职位 | 必需 |

### 4.3 高管/经理类别额外文件

| 文件 | 说明 |
|------|------|
| 管理职责证明 | 会议记录、决策文件 |
| 下属员工名单 | 直接汇报人员 |
| 签字权限证明 | 银行授权、合同签署权 |
| 战略规划文件 | 参与制定的证据 |

### 4.4 专业知识类别额外文件

| 文件 | 说明 |
|------|------|
| 培训记录 | 专业培训证书 |
| 项目文档 | 参与的专有项目 |
| 技术文档 | 掌握的专有技术 |
| 专利/知识产权 | 如适用 |

### 4.5 新成立公司额外文件

根据案例法（Babalou 2024 FC 549）要求：

| 文件 | 说明 | 重要性 |
|------|------|--------|
| 商业计划书 | 详细的运营计划 | 必需 |
| 财务预测 | 收入和支出预测 | 必需 |
| 客户合同/意向书 | 证明业务可行性 | 强烈建议 |
| 办公场所证明 | 租约或购买协议 | 必需 |
| 招聘计划 | 本地员工招聘计划 | 必需 |
| 市场分析 | 加拿大市场研究 | 建议 |

## 五、配偶开放工签文件清单

### 5.1 关系证明文件

| 文件 | 说明 | 重要性 |
|------|------|--------|
| 结婚证 | 公证翻译件 | 必需（已婚） |
| 同居声明 | IMM 5409 | 必需（同居） |
| 关系时间线 | 认识、交往、结婚日期 | 强烈建议 |

### 5.2 共同居住证明

**至少提供2类：**

| 类型 | 示例 |
|------|------|
| 住所证明 | 联名租约、房产证 |
| 公用事业 | 联名水电费账单 |
| 财务证明 | 联名银行账户 |
| 政府文件 | 同地址驾照、医疗卡 |
| 保险 | 联名汽车/健康保险 |

### 5.3 主申请人文件

| 主申请人类型 | 需要的文件 |
|--------------|-----------|
| 工签持有人 | 工作许可副本、雇佣证明、NOC确认 |
| 学签持有人 | 学习许可副本、全日制注册证明 |
| PR申请人 | AOR确认信、担保申请副本 |

### 5.4 关系真实性证据

根据配偶担保审核标准：

| 证据类型 | 权重 | 数量建议 |
|----------|------|----------|
| 共同居住证明 | 高 | 至少2类 |
| 财务捆绑证明 | 高 | 至少1类 |
| 沟通记录 | 中 | 最多10页 |
| 照片 | 低 | 最多20张 |
| 第三方证明信 | 低 | 2-3封 |

### 5.5 照片要求

**照片应展示：**
- 不同时间拍摄（跨越数月/年）
- 不同地点（多个场所）
- 包含家人/朋友
- 重要事件（婚礼、节日、旅行）

**每张照片标注：**
- 拍摄日期
- 拍摄地点
- 照片中人物

## 七、文件准备通用指南

### 7.1 翻译要求

| 文件类型 | 翻译要求 |
|----------|----------|
| 非英文/法文文件 | 必须附公证翻译 |
| 翻译件 | 需包含翻译员声明 |
| 原件 | 同时提交原件和翻译件 |

**翻译员声明应包含：**
- 翻译员姓名和资质
- 翻译日期
- 声明翻译准确完整
- 翻译员签名

### 7.2 文件格式要求

| 要求 | 说明 |
|------|------|
| 文件格式 | PDF（在线申请） |
| 文件大小 | 每个文件不超过4MB |
| 扫描质量 | 清晰可读，彩色扫描 |
| 文件命名 | 清晰描述内容 |

### 7.3 文件组织建议

**建议的文件夹结构：**
```
申请文件/
├── 01_身份文件/
│   ├── 护照.pdf
│   ├── 出生证明.pdf
│   └── 结婚证.pdf
├── 02_申请表格/
│   ├── IMM1295.pdf
│   └── IMM5707.pdf
├── 03_工作文件/
│   ├── LMIA.pdf
│   ├── Offer_Letter.pdf
│   └── 雇佣合同.pdf
├── 04_资格证明/
│   ├── 学历证明.pdf
│   ├── 工作经验证明.pdf
│   └── 专业证书.pdf
└── 05_其他文件/
    ├── 照片.pdf
    └── 代理授权.pdf
```

### 7.4 常见错误避免

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 文件不完整 | 申请退回 | 使用清单逐项核对 |
| 翻译缺失 | 申请延迟 | 所有非英法文件附翻译 |
| 信息不一致 | 可信度受损 | 核对所有文件信息一致 |
| 文件过期 | 申请拒绝 | 检查所有文件有效期 |
| 扫描不清 | 要求重新提交 | 使用高质量扫描 |

| 文件类型 | 有效期要求 |
