# 工作许可个人陈述模板指南

**适用范围**: LMIA工作许可、PGWP、ICT公司内部调动、配偶开放工签
**语言**: 中文
**版本**: 1.0.0
**最后更新**: 2026年1月

---

## 一、文件基本信息

```
日期：[填写日期]
档案号：[填写档案号，如有]
申请人：[姓名] (出生日期: [YYYY-MM-DD])
申请类型：[LMIA工作许可 / PGWP / ICT / 配偶开放工签]
```

---

## 二、LMIA工作许可个人陈述

### A. 陈述结构

#### 1. 个人背景介绍（2-3段）

**必须包含：**
- 教育背景（学历、专业、毕业院校）
- 工作经历摘要（总年限、主要领域）
- 与申请职位相关的核心技能

**示例：**
```
我叫[姓名]，[国籍]公民，拥有[学校名称][专业]学位。过去[X]年，
我一直从事[行业/领域]工作，积累了丰富的[具体技能]经验。

我目前在[公司名称]担任[职位]，主要负责[核心职责]。在此期间，
我成功[具体成就，如完成项目、提升业绩等]。
```

#### 2. 为何选择加拿大雇主（2-3段）

**必须包含：**
- 如何了解到该职位/公司
- 该公司/职位的吸引力
- 为何认为自己是合适人选

**示例：**
```
我通过[招聘渠道/推荐人]了解到[加拿大公司名称]正在招聘[职位]。
该公司在[行业]领域的领先地位和[具体特点]深深吸引了我。

我的[具体技能/经验]与该职位要求高度匹配。特别是我在[具体领域]
的经验，能够帮助公司[具体贡献]。
```

#### 3. 工作计划与贡献（2-3段）

**必须包含：**
- 对职位的理解
- 计划如何开展工作
- 预期为公司带来的价值

**示例：**
```
作为[职位]，我计划在入职后首先[短期计划]，然后逐步[中期目标]。
我相信我的[具体技能]能够帮助团队[具体贡献]。

长期来看，我希望能够[长期目标]，为公司的[业务方向]做出贡献。
```

#### 4. 临时性意图说明（1-2段）

**必须包含：**
- 明确表示此次来加是临时性的
- 工作许可到期后的计划
- 与母国的联系

**示例：**
```
我理解此工作许可是临时性的。工作许可到期后，我计划返回[母国]
继续我的职业发展。我在[母国]有[家庭/房产/工作机会等]，这些
都是我回国的重要纽带。
```

### B. LMIA工作许可注意事项

| 要点 | 说明 |
|------|------|
| 职位匹配 | 陈述内容必须与LMIA批准的职位一致 |
| 工资水平 | 不要提及低于LMIA规定的工资 |
| 工作地点 | 必须与LMIA批准的地点一致 |
| 雇主信息 | 确保公司名称、地址准确无误 |

## 三、PGWP毕业工签个人陈述

### A. 陈述结构

#### 1. 学业完成情况（2-3段）

**必须包含：**
- 就读学校（必须是DLI）
- 专业和学位类型
- 学习时长和完成日期
- 全日制学习状态确认

**示例：**
```
我于[入学日期]进入[学校名称]（DLI编号：[编号]）攻读[专业名称]
[学位类型]。该项目为期[X]个月/年，我于[完成日期]顺利完成
全部学业并获得毕业证书。

在整个学习期间，我保持全日制学生身份，每学期修读[X]门课程，
符合学校的全日制学习要求。
```

#### 2. 学习期间经历（2-3段）

**必须包含：**
- 主要学习内容和技能
- 实习/Co-op经历（如有）
- 学术成就或项目经验

**示例：**
```
在学习期间，我系统掌握了[核心课程/技能]。特别是在[具体领域]
方面，我通过[课程/项目]获得了深入的理论知识和实践经验。

我还参与了学校的Co-op项目，在[公司名称]实习[X]个月，担任
[职位]，负责[具体工作内容]。这段经历让我对加拿大职场有了
更深入的了解。
```

#### 3. 职业规划（2-3段）

**必须包含：**
- 希望从事的工作类型
- 如何运用所学知识
- 职业发展目标

**示例：**
```
毕业后，我希望在[行业/领域]寻找与我专业相关的工作机会。
我的[专业技能]和[实习经验]使我能够胜任[目标职位类型]。

我计划在PGWP期间积累加拿大工作经验，提升[具体技能]，
为未来的职业发展打下坚实基础。
```

### B. PGWP关键时间节点

| 时间节点 | 要求 | 风险等级 |
|----------|------|----------|
| 申请截止 | 最终成绩发布后180天内 | 致命 |
| 学习许可有效期 | 申请时必须在180天窗口内有效过 | 致命 |
| 项目时长 | 最少8个月 | 致命 |
| 全日制状态 | 整个学习期间保持 | 高 |

### C. PGWP特殊情况说明

**如有以下情况，必须在陈述中解释：**

1. **学习中断**
```
由于[具体原因，如医疗原因]，我在[时间段]期间暂时中断学习。
我已附上[医疗证明/学校确认信]作为支持文件。学校已确认这
不影响我的全日制学生身份。
```

2. **项目变更**
```
我最初就读于[原项目]，后于[日期]转入[新项目]。两个项目
均为全日制学术项目，总学习时长为[X]个月，符合PGWP要求。
```

3. **学习许可过期**
```
我的学习许可于[日期]过期，但我在过期后90天内（[申请日期]）
提交了PGWP申请，并同时支付了身份恢复费用。
```

## 四、ICT公司内部调动个人陈述

### A. 陈述结构

#### 1. 母公司工作经历（3-4段）

**必须包含：**
- 母公司基本信息（名称、行业、规模）
- 入职时间和职位发展
- 具体工作职责和成就
- 连续工作满1年的证明

**示例：**
```
我自[入职日期]起在[母公司名称]工作，至今已连续工作[X]年。
[母公司名称]是一家[行业]领域的[跨国/国际]企业，在全球
[X]个国家设有分支机构，员工总数超过[X]人。

我目前担任[职位]，直接向[上级职位]汇报。我的主要职责包括：
- [职责1]
- [职责2]
- [职责3]

在过去[X]年中，我成功[具体成就]，为公司[具体贡献]。
```

#### 2. 公司关系说明（2-3段）

**必须包含：**
- 母公司与加拿大公司的关系（母子公司/分支/关联公司）
- 股权结构或控制关系
- 两家公司的业务关联

**示例：**
```
[加拿大公司名称]是[母公司名称]的[全资子公司/分支机构/关联公司]。
[母公司]持有[加拿大公司][X]%的股份，对其拥有[完全/主要]控制权。

两家公司在业务上紧密合作，[加拿大公司]主要负责[业务范围]，
与母公司的[业务领域]形成协同效应。
```

#### 3. 调动原因和职位说明（2-3段）

**必须包含：**
- 为何需要调动到加拿大
- 加拿大职位的具体职责
- 如何运用专业知识/管理经验

**示例（高管/经理类）：**
```
公司决定派遣我到加拿大分公司担任[职位]，主要负责[核心职责]。
作为[高管/经理]，我将：
- 管理[X]名员工的团队
- 制定和执行[业务/部门]战略
- 对[具体业务领域]拥有决策权

我在母公司积累的[管理经验/行业知识]将帮助加拿大分公司
[具体目标，如拓展市场、提升效率等]。
```

**示例（专业知识类）：**
```
公司派遣我到加拿大是因为我拥有[具体专业知识]，这是公司
[产品/服务/流程]的核心技术，需要由经过专门培训的员工来
实施和传授。

我在母公司接受了[X]年的专业培训，掌握了[具体技术/流程]。
这些知识是公司专有的，无法在加拿大劳动力市场上轻易获得。
```

#### 4. 临时性意图（1-2段）

**必须包含：**
- 调动的临时性质
- 预计在加拿大的工作期限
- 调动结束后的计划

**示例：**
```
此次调动是临时性的，预计为期[X]年。调动期间，我将完成
[具体任务/目标]。任务完成后，我计划返回[母公司所在地]
继续我的职业发展。

我在[母国]有[家庭/房产/职位保留等]，这些都是我回国的
重要纽带。
```

### B. ICT类别特定要求

| 类别 | 职位要求 | 最长期限 |
|------|----------|----------|
| 高管 | 指导公司/主要部门的管理 | 7年 |
| 高级经理 | 管理组织/部门/职能 | 7年 |
| 专业知识 | 拥有公司专有知识 | 5年 |

### C. ICT高风险因素

根据案例法（Chamma 2018 FC 29, Babalou 2024 FC 549）：

| 风险因素 | 说明 | 缓解措施 |
|----------|------|----------|
| 新成立公司 | 加拿大公司刚成立 | 提供详细商业计划和财务预测 |
| 无实际运营 | 公司无员工或业务 | 证明"正在开展业务" |
| 商业计划不切实际 | 收入预测无依据 | 提供客户合同或意向书 |
| 业务不匹配 | 母公司与加拿大公司业务不同 | 解释业务关联性 |
| 未披露拒签史 | 隐瞒之前的拒签 | 完整披露所有申请历史 |

## 五、配偶开放工签个人陈述

### A. 陈述结构

#### 1. 关系说明（3-4段）

**必须包含：**
- 与主申请人的关系（配偶/同居伴侣）
- 认识和交往经过
- 结婚/同居时间
- 目前的共同生活状况

**示例：**
```
我是[主申请人姓名]的[配偶/同居伴侣]。我们于[日期]在[地点]
通过[方式]相识，经过[X]个月/年的交往，于[日期]在[地点]
正式结婚/开始同居。

我们目前共同居住在[地址]，已经一起生活了[X]年。我们共同
承担家庭开支，包括[租金/房贷、水电费等]。
```

#### 2. 主申请人情况（2-3段）

**必须包含：**
- 主申请人的身份（工签持有人/学签持有人/PR申请人）
- 主申请人的工作/学习情况
- 主申请人许可证的有效期

**示例（配偶为工签持有人）：**
```
我的配偶[姓名]目前持有有效的工作许可，在[公司名称]担任
[职位]。该职位属于NOC [编号]（TEER [0/1/2/3]级别），
工作许可有效期至[日期]。

[配偶姓名]在该公司工作已有[X]年，负责[主要职责]。
```

**示例（配偶为学签持有人）：**
```
我的配偶[姓名]目前在[学校名称]（DLI）攻读[学位类型][专业]。
这是一个全日制[硕士/博士/专业学位]项目，学习许可有效期至[日期]。
```

**示例（正在申请PR）：**
```
我的配偶[姓名]是加拿大公民/永久居民，目前正在担保我申请
永久居民身份。我们的担保申请已于[日期]被确认为完整（AOR），
目前正在处理中。
```

#### 3. 申请开放工签的原因（1-2段）

**必须包含：**
- 为何需要工作
- 计划从事的工作类型
- 如何支持家庭

**示例：**
```
我申请开放工签是为了能够在加拿大合法工作，支持我们的家庭
生活。我拥有[教育背景/工作经验]，希望能够在[行业/领域]
找到工作机会。

工作收入将帮助我们支付[生活费用/学费/房租等]，减轻家庭
经济压力。
```

### B. 关系真实性证据清单

根据配偶担保审核标准，开放工签同样需要证明关系真实性：

| 证据类型 | 权重 | 示例 |
|----------|------|------|
| 共同居住证明 | 高 | 联名租约、共同地址的政府文件 |
| 财务捆绑证明 | 高 | 联名银行账户、共同账单 |
| 沟通记录 | 中 | 聊天记录、通话记录 |
| 照片 | 低 | 不同时间地点的合照 |
| 第三方证明 | 低 | 亲友证明信 |

### C. 配偶工签类别

| 类别 | 代码 | 主申请人要求 |
|------|------|--------------|
| 技术工人配偶 | C41 | 持有NOC TEER 0/1/2/3工签 |
| 学生配偶 | C42 | 全日制硕士/博士/专业学位学生 |
| PR担保配偶 | C46 | 已提交完整的担保申请 |

## 六、通用写作原则

### A. 语言风格

| 要求 | 说明 |
|------|------|
| 正式但不僵硬 | 使用书面语，避免口语化表达 |
| 具体而非笼统 | 提供具体日期、数字、事例 |
| 诚实准确 | 所有陈述必须真实，可被证据支持 |
| 逻辑清晰 | 按时间顺序或逻辑顺序组织内容 |

### B. 篇幅建议

| 申请类型 | 建议篇幅 |
|----------|----------|
| LMIA工作许可 | 1000-1500字 |
| PGWP | 800-1200字 |
| ICT | 1500-2000字 |
| 配偶开放工签 | 1000-1500字 |

### C. 避免事项

- 不要夸大或虚构经历
- 不要使用模板化语言
- 不要遗漏重要信息
- 不要与其他申请文件矛盾
- 不要包含无法证明的陈述
