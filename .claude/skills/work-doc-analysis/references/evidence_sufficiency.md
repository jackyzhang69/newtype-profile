# 工作许可证据充分性标准

**适用范围**: LMIA工作许可、PGWP、ICT、配偶开放工签、PNP工签
**版本**: 1.0.0
**最后更新**: 2026年1月

---

## 一、证据充分性评估框架

### 1.1 评估维度

| 维度 | 描述 | 权重 |
|------|------|------|
| **完整性** | 是否提供了所有必需文件 | 40% |
| **质量** | 文件等级分布（A/B/C/D） | 30% |
| **一致性** | 文件间信息是否一致 | 20% |
| **时效性** | 文件是否在有效期内 | 10% |

### 1.2 充分性等级

| 等级 | 分数范围 | 描述 | 建议 |
|------|----------|------|------|
| **优秀** | 90-100 | 证据充分，质量高 | 可以提交 |
| **良好** | 75-89 | 证据基本充分 | 建议补充 |
| **可接受** | 60-74 | 证据勉强充分 | 需要补充 |
| **不足** | <60 | 证据不充分 | 不建议提交 |

## 二、各类别充分性标准

### 2.1 LMIA 工作许可

#### 最低要求清单

| 证据类别 | 最低要求 | 理想状态 |
|----------|----------|----------|
| **核心证据** | LMIA + Offer + 护照 | 全部A级 |
| **资格证明** | 学历证书 | + 认证 + 专业证书 |
| **经验证明** | 1封推荐信 | + 工资单 + 税单 |

#### 充分性评分表

```
LMIA工作许可证据评分:
┌─────────────────────────────────────────────────────────┐
│  核心证据 (40分)                                        │
│  ├─ LMIA批准信: 15分 (必需)                             │
│  ├─ 雇主Offer: 15分 (必需)                              │
│  └─ 护照: 10分 (必需)                                   │
│                                                         │
│  资格证明 (30分)                                        │
│  ├─ 学历证书: 15分 (必需)                               │
│  ├─ 学历认证: 10分 (建议)                               │
│  └─ 专业证书: 5分 (如适用)                              │
│                                                         │
│  经验证明 (30分)                                        │
│  ├─ 推荐信: 15分 (必需)                                 │
│  ├─ 工资单: 10分 (建议)                                 │
│  └─ 税单: 5分 (建议)                                    │
│                                                         │
│  总分: 100分                                            │
│  最低通过: 60分 (必需项全部满足)                        │
└─────────────────────────────────────────────────────────┘
```

#### 常见不足情况

| 不足类型 | 影响 | 补救措施 |
|----------|------|----------|
| 无学历认证 | -10分 | 提交原件 + 翻译 |
| 推荐信不完整 | -5~15分 | 补充缺失信息 |
| 无工资单 | -10分 | 提供税单或银行流水 |

#### 关键验证点

| 验证点 | 通过标准 | 失败后果 |
|--------|----------|----------|
| DLI状态 | 学校在DLI列表 | 申请被拒 |
| 项目时长 | ≥8个月 | 申请被拒 |
| 申请时间 | 180天内 | 申请被拒 |
| 全日制状态 | 全程保持 | 需要解释 |

## 三、证据缺口分析

### 3.1 缺口识别

| 缺口类型 | 识别方法 | 严重程度 |
|----------|----------|----------|
| **必需文件缺失** | 清单对照 | 致命 |
| **信息不完整** | 内容检查 | 高 |
| **质量不足** | 等级评估 | 中 |
| **时效过期** | 日期检查 | 中-高 |

### 3.2 缺口补救

| 缺口类型 | 补救措施 | 时间要求 |
|----------|----------|----------|
| 文件缺失 | 获取原件或替代 | 申请前 |
| 信息不完整 | 补充说明信 | 申请时 |
| 质量不足 | 升级证据等级 | 申请前 |
| 时效过期 | 获取新文件 | 申请前 |

### 3.3 解释信要求

当存在以下情况时，必须提供解释信：

- [ ] 无法提供必需文件
- [ ] 文件信息不一致
- [ ] 存在时间线空白
- [ ] 特殊情况需要说明

**解释信质量标准：**
- 具体说明原因
- 提供替代证据
- 逻辑一致
- 语气真诚

## 四、证据充分性检查清单

### 4.1 通用检查

- [ ] 所有必需文件已提供
- [ ] 文件在有效期内
- [ ] 信息跨文件一致
- [ ] 翻译件已附（如需）
- [ ] 文件格式正确（PDF）
- [ ] 文件清晰可读

### 4.2 类别特定检查

**LMIA:**
- [ ] LMIA编号可验证
- [ ] 职位/工资与LMIA一致
- [ ] 学历满足要求
- [ ] 工作经验充分

**PGWP:**
- [ ] DLI状态确认
- [ ] 180天窗口内
- [ ] 全日制状态确认
- [ ] 项目时长≥8个月

**ICT:**
- [ ] 公司关系明确
- [ ] 连续工作1年+
- [ ] 职位符合定义
- [ ] 新公司有商业计划

**配偶:**
- [ ] 关系证明充分
- [ ] 主申请人资格确认
- [ ] 共同居住证据≥2类
- [ ] 无真实性红旗

**PNP:**
- [ ] 提名仍然有效
- [ ] 雇主仍在运营
- [ ] 保持雇主联系
- [ ] 无虚假陈述风险
