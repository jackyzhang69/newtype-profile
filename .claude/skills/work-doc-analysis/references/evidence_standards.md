# 工作许可证据质量标准

**适用范围**: LMIA工作许可、PGWP、ICT、配偶开放工签、PNP工签
**版本**: 1.0.0
**最后更新**: 2026年1月

---

## 一、证据质量评估框架

### 1.1 证据权重分级

| 级别 | 描述 | 工作许可示例 |
|------|------|-------------|
| **A级 (最高)** | 政府签发、难以伪造、直接证明资格 | LMIA批准信、学位证书、省提名证书、工作许可 |
| **B级 (高)** | 官方机构签发、有日期戳、可验证 | 雇佣合同、银行工资单、税单、公司注册文件 |
| **C级 (中)** | 第三方文件、有日期、可追溯 | 雇主推荐信、学校成绩单、培训证书 |
| **D级 (低)** | 自制文件、难以验证 | 个人陈述、组织架构图、商业计划书 |

### 1.2 证据充分性标准

```
理想证据组合:
┌─────────────────────────────────────────────────────────┐
│  核心证据 (必需)                                        │
│  ├─ 资格证明 (A级)                                      │
│  ├─ 雇主/学校文件 (A/B级)                               │
│  └─ 身份文件 (A级)                                      │
│                                                         │
│  支持证据 (强烈建议)                                    │
│  ├─ 工作经验证明 (B/C级)                                │
│  ├─ 财务能力证明 (B级)                                  │
│  └─ 临时性意图证明 (C/D级)                              │
│                                                         │
│  补充证据 (根据情况)                                    │
│  ├─ 专业资质证明                                        │
│  ├─ 语言能力证明                                        │
│  └─ 与母国联系证明                                      │
└─────────────────────────────────────────────────────────┘
```

## 二、各类别证据评估标准

### 2.1 LMIA 工作许可证据

#### A级证据 (必需)

| 证据类型 | 来源 | 验证方法 |
|----------|------|----------|
| LMIA批准信 | ESDC | LMIA编号可验证 |
| 护照 | 政府签发 | 有效期、签名页 |
| 学位证书 | 教育机构 | 可通过学校验证 |

#### B级证据 (强烈建议)

| 证据类型 | 来源 | 质量指标 |
|----------|------|----------|
| 雇佣合同 | 雇主 | 包含职位、工资、条款 |
| 工资单 | 雇主/银行 | 连续12个月以上 |
| 税单 | 税务机关 | 与工资单一致 |
| 学历认证 | WES/IQAS等 | 官方认证报告 |

#### C级证据 (支持)

| 证据类型 | 来源 | 注意事项 |
|----------|------|----------|
| 推荐信 | 前雇主 | 需包含具体职责和日期 |
| 培训证书 | 培训机构 | 与职位相关 |
| 项目文档 | 雇主 | 证明专业能力 |

#### D级证据 (补充)

| 证据类型 | 用途 | 局限性 |
|----------|------|--------|
| 个人陈述 | 解释背景 | 主观性强 |
| 职业规划 | 说明意图 | 难以验证 |

#### LMIA证据质量检查清单

- [ ] LMIA编号与批准信一致
- [ ] 职位名称与LMIA一致
- [ ] 工资水平与LMIA一致
- [ ] 工作地点与LMIA一致
- [ ] 学历满足职位要求
- [ ] 工作经验满足要求

### 2.2 PGWP 毕业工签证据

#### A级证据 (必需)

| 证据类型 | 来源 | 关键验证点 |
|----------|------|-----------|
| 学校完成信 | DLI注册处 | DLI编号、完成日期 |
| 学习许可 | IRCC | 有效期覆盖180天窗口 |
| 护照 | 政府签发 | 有效期 |

#### B级证据 (强烈建议)

| 证据类型 | 来源 | 质量指标 |
|----------|------|----------|
| 正式成绩单 | 学校 | 显示所有课程和成绩 |
| 毕业证书 | 学校 | 学位类型和专业 |

#### C级证据 (特殊情况需要)

| 证据类型 | 适用情况 | 说明 |
|----------|----------|------|
| 医疗证明 | 学习中断 | 解释非全日制原因 |
| 学校确认信 | 项目变更 | 确认学习状态 |
| 身份恢复收据 | 学签过期 | $350恢复费 |

#### PGWP证据质量检查清单

- [ ] 学校是DLI（检查DLI列表）
- [ ] 项目时长≥8个月
- [ ] 全日制学习状态确认
- [ ] 最终成绩发布日期明确
- [ ] 申请在180天内提交
- [ ] 学习许可在窗口内有效过

#### PGWP时间线验证

```
关键时间点:
┌────────────────────────────────────────────────────────┐
│  入学日期      最终成绩发布      申请截止              │
│  ──●──────────────●──────────────●──                   │
│    │              │              │                     │
│    项目开始       180天窗口开始   180天窗口结束        │
│                                                        │
│  学习许可必须在180天窗口内某一时点有效                │
└────────────────────────────────────────────────────────┘
```

### 2.3 ICT 公司内部调动证据

#### A级证据 (必需)

| 证据类型 | 来源 | 关键验证点 |
|----------|------|-----------|
| 母公司注册文件 | 政府 | 公司合法存在 |
| 加拿大公司注册文件 | 政府 | 公司合法存在 |
| 股权结构证明 | 公司 | 母子公司关系 |
| 护照 | 政府签发 | 有效期 |

#### B级证据 (必需)

| 证据类型 | 来源 | 质量指标 |
|----------|------|----------|
| 雇佣证明信 | 母公司 | 职位、日期、职责 |
| 工资单 | 母公司 | 连续12个月以上 |
| 税单 | 税务机关 | 证明收入 |
| 组织架构图 | 公司 | 显示职位和汇报关系 |

#### C级证据 (类别特定)

**高管/经理类别:**

| 证据类型 | 用途 |
|----------|------|
| 会议记录 | 证明决策参与 |
| 签字权限证明 | 证明管理权限 |
| 下属员工名单 | 证明管理职责 |
| 战略规划文件 | 证明高层参与 |

**专业知识类别:**

| 证据类型 | 用途 |
|----------|------|
| 培训记录 | 证明专业培训 |
| 项目文档 | 证明专有知识应用 |
| 技术文档 | 证明专业技能 |
| 专利/知识产权 | 证明独特贡献 |

#### D级证据 (新公司必需)

| 证据类型 | 用途 | 质量要求 |
|----------|------|----------|
| 商业计划书 | 证明业务可行性 | 详细、现实、有依据 |
| 财务预测 | 证明收入预期 | 有客户/合同支持 |
| 市场分析 | 证明市场需求 | 基于实际研究 |
| 招聘计划 | 证明本地雇佣意图 | 具体职位和时间表 |

#### ICT证据质量检查清单

- [ ] 母子公司关系明确证明
- [ ] 两家公司都"正在开展业务"
- [ ] 连续工作满1年（过去3年内）
- [ ] 职位符合高管/经理/专业知识定义
- [ ] 工资符合加拿大标准
- [ ] 临时性意图有证据支持

#### ICT高风险情况额外证据

根据案例法（Chamma 2018 FC 29, Babalou 2024 FC 549）：

| 风险情况 | 额外证据要求 |
|----------|-------------|
| 新成立公司 | 详细商业计划 + 客户合同/意向书 |
| 业务不匹配 | 解释信 + 业务关联证明 |
| 家庭在加拿大 | 临时性意图证明 + 回国计划 |
| 之前拒签 | 完整披露 + 情况变化说明 |

### 2.4 配偶开放工签证据

#### A级证据 (必需)

| 证据类型 | 来源 | 关键验证点 |
|----------|------|-----------|
| 结婚证/同居声明 | 政府/公证 | 关系法律状态 |
| 主申请人许可证 | IRCC | 有效期、类型 |
| 护照 | 政府签发 | 有效期 |

#### B级证据 (关系证明)

| 证据类型 | 权重 | 最低要求 |
|----------|------|----------|
| 共同居住证明 | 高 | 至少2类 |
| 财务捆绑证明 | 高 | 至少1类 |
| 主申请人雇佣证明 | 高 | NOC确认 |

#### C级证据 (支持)

| 证据类型 | 用途 |
|----------|------|
| 沟通记录 | 证明关系持续 |
| 照片 | 证明关系真实 |
| 第三方证明信 | 社会认可 |

#### 配偶工签证据质量检查清单

**主申请人验证:**
- [ ] 工作许可/学习许可有效
- [ ] NOC TEER 0/1/2/3（C41）
- [ ] 全日制硕士/博士/专业学位（C42）
- [ ] 许可有效期≥6个月

**关系真实性验证:**
- [ ] 关系时间线合理
- [ ] 共同居住证据充分
- [ ] 无便利婚姻红旗
- [ ] 与配偶担保标准一致

#### 与配偶担保标准对齐

配偶开放工签的关系真实性评估参照配偶担保标准：

| 风险模式 | 来源 | 适用性 |
|----------|------|--------|
| rushed_relationship | spousal-audit-rules | 完全适用 |
| lack_cohabitation_evidence | spousal-audit-rules | 完全适用 |
| immigration_motive | spousal-audit-rules | 完全适用 |
| testimony_contradiction | spousal-audit-rules | 完全适用 |

### 2.5 PNP 省提名工签证据

#### A级证据 (必需)

| 证据类型 | 来源 | 关键验证点 |
|----------|------|-----------|
| 省提名证书 | 省政府 | 有效期、提名编号 |
| 联邦申请确认 | IRCC | AOR或提交证明 |
| 护照 | 政府签发 | 有效期 |

#### B级证据 (必需)

| 证据类型 | 来源 | 质量指标 |
|----------|------|----------|
| 雇主Offer Letter | 雇主 | 与提名一致 |
| 雇佣合同 | 雇主 | 详细条款 |
| 雇主确认信 | 雇主 | 近期出具 |

#### C级证据 (雇主合法性)

根据案例法（Baniya 2022 FC 18）：

| 证据类型 | 用途 | 重要性 |
|----------|------|--------|
| 公司注册文件 | 证明公司存在 | 高 |
| 营业执照 | 证明合法运营 | 高 |
| 公司网站截图 | 证明活跃运营 | 中 |
| 办公场所照片 | 证明实际存在 | 中 |
| 最近通讯记录 | 证明持续联系 | 高 |

#### PNP证据质量检查清单

- [ ] 省提名仍然有效
- [ ] 联邦申请已提交
- [ ] 工作offer仍然有效
- [ ] 雇主仍在运营
- [ ] 与雇主保持联系
- [ ] 无虚假陈述风险

#### PNP高风险情况

| 风险情况 | 证据要求 |
|----------|----------|
| 雇主不回应 | 联系尝试记录 |
| 公司地址变更 | 新地址验证 |
| 长时间处理 | 定期雇主确认 |
| 雇主被调查 | 考虑替代方案 |

## 三、证据时间线分析

### 3.1 各类别时间线要求

| 类别 | 关键时间点 | 证据覆盖要求 |
|------|-----------|-------------|
| LMIA | LMIA有效期内 | 雇佣开始前 |
| PGWP | 180天窗口 | 学习全程 |
| ICT | 过去3年内1年 | 连续雇佣证明 |
| 配偶 | 关系发展全程 | 每阶段有证据 |
| PNP | 提名至PR | 雇主持续有效 |

### 3.2 时间线红旗检测

| 红旗 | 描述 | 严重程度 |
|------|------|---------|
| 证据空白期 | 关键时期无证据 | 高 |
| 时间不匹配 | 文件日期与陈述矛盾 | 致命 |
| 申请前集中 | 证据集中在申请前 | 中 |
| 雇主失联 | 长期无法联系雇主 | 高 |

## 四、证据真实性验证

### 4.1 文件真实性检查

| 检查项 | 方法 |
|--------|------|
| LMIA | 验证LMIA编号 |
| 学校文件 | 核对DLI列表、联系学校 |
| 公司文件 | 公司注册查询 |
| 银行文件 | 检查格式、账号、日期 |

### 4.2 内部一致性检查

- [ ] 姓名拼写一致
- [ ] 日期无矛盾
- [ ] 职位/职责描述一致
- [ ] 工资/收入信息一致
- [ ] 地址信息一致

### 4.3 外部一致性检查

- [ ] 与申请表信息一致
- [ ] 与面试陈述一致
- [ ] 与之前申请一致
- [ ] 与公开信息一致

## 五、证据缺口处理

### 5.1 可接受的缺口解释

| 缺口类型 | 可接受解释 |
|----------|-----------|
| 工资单缺失 | 现金支付行业 + 税单证明 |
| 推荐信缺失 | 公司倒闭 + 替代证明 |
| 学历认证缺失 | 时间紧迫 + 原件 + 承诺补交 |
| 雇主文件缺失 | 公司重组 + 替代证明 |

### 5.2 必须提供解释信的情况

- [ ] 无法提供核心证据
- [ ] 证据存在明显缺口
- [ ] 信息存在不一致
- [ ] 特殊情况需要说明

### 5.3 解释信质量标准

- [ ] 具体说明原因
- [ ] 提供替代证据
- [ ] 前后逻辑一致
- [ ] 语气真诚合理
- [ ] 附支持文件

## 六、证据评分矩阵

### 6.1 LMIA工作许可

| 证据类型 | 权重 | 最低要求 | 理想状态 |
|----------|------|----------|----------|
| LMIA批准信 | 100% | 有效原件 | - |
| 雇主Offer | 90% | 基本信息 | 详细条款 |
| 学历证明 | 80% | 学位证书 | + 认证报告 |
| 工作经验 | 70% | 推荐信 | + 工资单 + 税单 |
| 专业证书 | 50% | 相关证书 | 多项认证 |

### 6.2 PGWP

| 证据类型 | 权重 | 最低要求 | 理想状态 |
|----------|------|----------|----------|
| 完成信 | 100% | DLI出具 | 详细信息 |
| 成绩单 | 90% | 正式成绩单 | 全程记录 |
| 学习许可 | 100% | 有效副本 | - |
| 全日制证明 | 80% | 学校确认 | 每学期记录 |

### 6.3 ICT

| 证据类型 | 权重 | 最低要求 | 理想状态 |
|----------|------|----------|----------|
| 公司关系证明 | 100% | 股权文件 | + 注册文件 |
| 雇佣证明 | 100% | 雇佣信 | + 工资单 + 税单 |
| 职位证明 | 90% | 职位描述 | + 组织架构 |
