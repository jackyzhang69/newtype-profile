# 配偶担保资格检查规则

## 一、担保人资格 (Sponsor Eligibility)

### 1.1 身份要求 (IRPR 130)

| 条件 | 要求 | 致命性 |
|------|------|--------|
| 身份状态 | 加拿大公民或永久居民 | Fatal |
| 年龄 | 18岁或以上 | Fatal |
| 居住要求 | 公民海外担保需证明将与配偶在加拿大居住 | Fatal |

### 1.2 不合格情况 (IRPR 133)

以下情况担保人**不具备资格**：

- [ ] 正在领取社会救助（非残疾原因）
- [ ] 未履行之前的担保承诺
- [ ] 未履行法院判定的赡养费或子女抚养费
- [ ] 处于破产程序中且未解除
- [ ] 曾因暴力犯罪被定罪（特定情况）
- [ ] 曾被遣返出加拿大
- [ ] 被拘留在监狱、监狱或改造设施中

### 1.3 担保冷却期

| 情况 | 冷却期 |
|------|--------|
| 作为配偶被担保获得PR | 5年内不能担保新配偶 |
| 之前担保的配偶关系 | 上次担保承诺期满后方可 |

## 二、申请人资格 (Applicant Eligibility)

### 2.1 关系类型 (IRPR 2, 117)

#### 配偶 (Spouse)
- [ ] 法定婚姻关系
- [ ] 婚姻在举行地合法
- [ ] 婚姻在加拿大法律下被承认
- [ ] 双方在结婚时均有结婚能力（无现有婚姻）

#### 同居伴侣 (Common-Law Partner)
- [ ] 连续共同居住至少12个月
- [ ] 关系具有婚姻性质（conjugal relationship）
- [ ] 任何分居期间关系仍保持持续性

#### 伴侣关系 (Conjugal Partner) - 特殊情况
- [ ] 至少维持一年的伴侣关系
- [ ] 因法律或移民障碍无法同居或结婚
- [ ] 需证明障碍的存在和性质

### 2.2 关系排除条件 (IRPR 4)

关系**不被承认**如果：

- [ ] 主要目的是获得移民身份或特权 - `IRPR 4(1)(a)`
- [ ] 关系不真实 - `IRPR 4(1)(b)`

**注意**：这是"或"测试，满足任一条件即排除。

### 2.3 申请人不合格情况

- [ ] 申请人已是加拿大公民或永久居民
- [ ] 申请人处于遣返令下
- [ ] 申请人不可入境（犯罪、安全、健康原因）
- [ ] 申请人此前因关系不真实被拒

## 三、关系真实性评估框架

### 3.1 核心评估维度

```
┌─────────────────────────────────────────────────────────┐
│                   关系真实性评估                        │
├─────────────────────────────────────────────────────────┤
│  1. 关系起源与发展                                      │
│     - 相识方式是否合理                                  │
│     - 发展时间线是否自然                                │
│     - 双方背景是否匹配                                  │
│                                                         │
│  2. 共同生活证据                                        │
│     - 财务捆绑程度                                      │
│     - 居住安排                                          │
│     - 日常互动证据                                      │
│                                                         │
│  3. 社会认可                                            │
│     - 家庭成员知悉和认可                                │
│     - 朋友圈了解                                        │
│     - 公开展示关系                                      │
│                                                         │
│  4. 未来规划                                            │
│     - 共同目标一致性                                    │
│     - 子女计划                                          │
│     - 长期居住安排                                      │
│                                                         │
│  5. 证词一致性                                          │
│     - 双方陈述一致                                      │
│     - 书面与口头一致                                    │
│     - 关键细节准确                                      │
└─────────────────────────────────────────────────────────┘
```

### 3.2 红旗清单 (Red Flags)

#### 致命红旗 (Fatal)
- 申请时机与遣返日期紧密相关
- 发现虚假文件
- 承认关系目的是移民
- 担保人有明确的虚假婚姻模式

#### 高风险红旗 (High)
- 首次见面到结婚 < 1个月
- 婚后从未见面
- 对配偶基本信息完全不了解
- 证词存在重大矛盾
- 隐瞒重要前婚史

#### 中等风险红旗 (Medium)
- 相识到结婚 < 3个月
- 婚礼无家人参加且解释不合理
- 沟通记录稀少或内容肤浅
- 年龄或文化差异大且无充分解释
- 财务支持方式异常

## 四、同居伴侣特殊规则

### 4.1 共同居住证明要求

**必须证明连续12个月共同居住**，可接受的证据：

| 类别 | 证据类型 | 权重 |
|------|---------|------|
| 住所 | 共同租赁协议、房产证 | 高 |
| 公用事业 | 共同水电费、网络账单 | 高 |
| 财务 | 共同银行账户、信用卡 | 高 |
| 政府文件 | 同地址驾照、税单 | 高 |
| 保险 | 共同汽车/健康保险 | 中 |
| 其他 | 同地址收到的邮件 | 低 |

### 4.2 分居期间处理

- 短暂分居（< 1个月）：需合理解释 + 保持联系证据
- 较长分居：需充分说明原因（工作、家庭紧急情况）+ 频繁互动证据
- 任何分居都不能中断关系的"共同生活性质"

## 五、文件完整性检查

### 5.1 担保人必需文件

- [ ] 身份证明（公民证/PR卡）
- [ ] 收入证明（NOA/Option C）
- [ ] 雇佣信（如适用）
- [ ] 前婚离婚证明（如适用）

### 5.2 申请人必需文件

- [ ] 护照/旅行证件
- [ ] 出生证明
- [ ] 婚姻证明
- [ ] 前婚离婚证明（如适用）
- [ ] 无犯罪记录证明
- [ ] 体检结果

### 5.3 关系证明文件 (IMM 5533 Part B.7)

- [ ] 共同居住证明（至少2类）
- [ ] 沟通记录（最多10页）
- [ ] 探访证明
- [ ] 照片（最多20张，不同时间地点）
- [ ] 亲友支持信

---

## 六、检查清单使用说明

1. 按顺序检查担保人资格 → 申请人资格 → 关系真实性
2. 任何 Fatal 项目不满足，停止评估并标记为不合格
3. High 风险项目需要详细记录和证据审查
4. 所有红旗项目需要在审核报告中说明
