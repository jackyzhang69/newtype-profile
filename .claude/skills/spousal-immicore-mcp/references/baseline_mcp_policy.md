# Baseline MCP Usage Policy

## Allowed Usage
- 仅在以下情况使用 MCP：
  - 用户要求深度分析/深度 audit。
  - Skill 覆盖不足（规则缺失、证据冲突无法裁决）。
- 优先语义搜索。

## Citation Requirements
- Include case name, citation, and year.
- Include manual name, section, and retrieval date.

## Failure Handling
- If MCP fails, state the limitation and proceed with local knowledge only.
- Do not fabricate citations or policy references.
