# Primary Assessment Template - Refugee Claims

## Quick Viability Check

Time target: 5-10 minutes

### 1. Eligibility Quick Check

**Hard stops (claim cannot proceed)**:
- [ ] Previous claim rejected/abandoned/withdrawn in Canada
- [ ] Already has protection status
- [ ] Inadmissible on serious grounds

**Warnings (needs further review)**:
- [ ] Entry from US (STCA may apply)
- [ ] Potential exclusion concerns
- [ ] Identity documentation gaps

### 2. Prima Facie Protection Assessment

**Convention ground identified**: ☐ Yes / ☐ No

If yes, which: _____________

**Persecution level**:
- [ ] Clear persecution described
- [ ] Discrimination only
- [ ] Unclear

**Agent of persecution**:
- [ ] State
- [ ] Non-state with state complicity
- [ ] Non-state only

### 3. Key Evidence Check

**Identity documents**: ☐ Present / ☐ Missing / ☐ Concerns

**Persecution evidence**: ☐ Present / ☐ Missing / ☐ Limited

**Country conditions**: ☐ Supports claim / ☐ Neutral / ☐ Contradicts

### 4. Red Flags

- [ ] Significant credibility concerns apparent
- [ ] Obvious IFA available
- [ ] State protection clearly adequate
- [ ] Exclusion grounds likely
- [ ] Long delay in claiming
