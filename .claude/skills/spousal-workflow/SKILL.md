# Spousal Workflow Templates

## Scope

- Define standardized output formats for each audit stage
- Ensure consistent report structure across assessments
- Guide agents in producing professional audit reports

## Workflow Stages

1. **Primary Assessment** - Initial risk identification and feasibility analysis
2. **Deep Analysis** - Detailed legal and evidence analysis
3. **Final Review** - Pre-submission compliance and consistency check
4. **Submission Letter** - Professional cover letter for IRCC submission

## Inputs

- Applicant and sponsor profile
- Collected evidence and documents
- Risk assessment results from Detective/Strategist

## Outputs

- Formatted assessment reports
- Checklists with actionable recommendations
- Professional submission letters

## Template Selection

| Stage | Template | Use Case |
|-------|----------|----------|
| Initial | `primary_assess_template.md` | First client meeting, risk overview |
| Analysis | `deep_analysis_template.md` | Detailed case evaluation |
| Pre-Submit | `final_assess_template.md` | Quality assurance before filing |
| Filing | `submission_letter_template.md` | IRCC cover letter |

## Notes

- Templates are in Chinese except submission letter (English)
- All templates include XML-style placeholders for structured data
- Reports should reference supporting evidence with exhibit numbers
- RCIC information should be included in submission letters
