# 学习许可文档清单生成模板

## 生成逻辑

根据申请人情况动态生成文档清单：

```
# SDS 已于 2024-11-08 关闭，以下材料现为"强烈推荐"而非强制
if wants_strong_application:
    recommend GIC, tuition_payment, language_score  # 原 SDS 要求，仍有帮助
if age < 18:
    include custody_declaration, parental_consent
if has_spouse:
    include spouse_documents
if province == "Quebec":
    include CAQ instead of PAL
```

## 清单模板

### 基础文件清单

```markdown
# [申请人姓名] 学习许可申请材料清单

## 必需文件 (Required)

### 身份文件
- [ ] 护照 (有效期至: YYYY-MM-DD)
- [ ] 护照照片 (2张, 35x45mm)
- [ ] 身份证复印件

### 录取文件
- [ ] 录取通知书 (DLI#: OXXXXXXXXX)
- [ ] 省级认证信 PAL (有效期至: YYYY-MM-DD)

### 财务文件
- [ ] 银行存款证明 (金额: $XXX,XXX)
- [ ] 银行对账单 (最近6个月)
- [ ] 担保人收入证明
- [ ] 资金来源说明信

### 教育文件
- [ ] 最高学历证书 + 翻译
- [ ] 成绩单 + 翻译
- [ ] 语言成绩单 (类型: IELTS/TOEFL, 分数: X.X)

### 申请表格
- [ ] IMM 1294
- [ ] IMM 5645/5707
- [ ] IMM 5257 (如需TRV)

## 支持性文件 (Supporting)

- [ ] 学习计划 (1-2页)
- [ ] 简历
- [ ] 工作证明/在职证明
- [ ] 房产证复印件
- [ ] 其他约束力证明
```

### ~~SDS 附加文件~~ → 强化材料建议（SDS 已关闭）

> **注意**: SDS 于 2024年11月8日关闭，以下材料不再强制，但仍为强有力的申请支持


## 文件状态标记

| 状态 | 标记 | 说明 |
|------|------|------|
| 已有 | ✅ | 文件已准备 |
| 缺失 | ❌ | 需要补充 |
| 待确认 | ⚠️ | 需要核实 |
| 不适用 | N/A | 本案不需要 |

## 文件准备提醒

### 有效期管理
| 文件 | 有效期 | 建议 |
|------|--------|------|
| 护照 | 学习期间+6个月 | 提前更新 |
| PAL | 90-120天 | 尽快使用 |
| 语言成绩 | 2年 | 1年内最佳 |
| 银行证明 | 3个月内 | 提交前更新 |

### 翻译要求
- 所有非英语/法语文件需认证翻译
- 翻译人员需提供资质证明
- 翻译件与原件一起提交
