# Executive Summary Template

## Structure

The executive summary provides a quick overview for decision-makers.

### Header Block

```
# REFUGEE PROTECTION CLAIM AUDIT REPORT
## Executive Summary

**Case ID**: [Case identifier]
**Claimant**: [Name] (DOB: [Date])
**Country of Origin**: [Country]
**Claim Date**: [Date claim submitted]
**Audit Date**: [Date of this report]
**Audit Tier**: [Guest / Pro / Ultra]
```

### Claim Overview

```
### Claim Overview

**Protection Ground**: [Convention Refugee under A96 / Person in Need of Protection under A97]
**Convention Ground** (if A96): [Race / Religion / Nationality / Political Opinion / Particular Social Group: specify]
**A97 Ground** (if applicable): [Torture / Risk to life / Cruel and unusual treatment]

**Agent of Persecution**: [State actor / Non-state actor / Both]
**Persecution Type**: [Brief description - e.g., political persecution, religious persecution]
```

## Writing Guidelines

### Tone
- Professional and objective
- Clear and concise
- Balanced (note both strengths and concerns)

### Length
- Executive summary should be 1 page maximum
- Use tables for quick scanning
- Detailed analysis in later sections

### Audience
- Immigration consultants
- Lawyers
- Claimants (with guidance)

### Key Principles
- Lead with the verdict
- Highlight critical issues immediately
- Be specific about concerns
- Provide actionable recommendations
