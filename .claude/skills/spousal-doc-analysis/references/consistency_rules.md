# 配偶担保跨文档一致性检查规则

---

## 一、核心一致性检查

### 1.1 身份信息一致性

| 字段 | 检查文件 | 常见问题 |
|------|---------|---------|
| 姓名拼写 | 护照、出生证、结婚证、申请表 | 音译差异、婚后改名未更新 |
| 出生日期 | 护照、出生证、IMM 0008 | 阴历/阳历混淆、格式错误 |
| 出生地点 | 护照、出生证 | 城市名称变更、详略不一 |
| 国籍 | 护照、申请表 | 双国籍声明不完整 |

### 1.2 关系时间线一致性

```
需要核对的关键日期:
┌─────────────────────────────────────────────────────────┐
│  日期类型           │  出现文件                        │
├─────────────────────┼──────────────────────────────────┤
│  首次相识日期       │  IMM 5532, Love Story, 支持信    │
│  开始交往日期       │  IMM 5532, Love Story            │
│  首次见面日期       │  IMM 5532, 机票/签证章           │
│  订婚日期           │  IMM 5532, 照片说明              │
│  结婚日期           │  结婚证, IMM 5532, IMM 0008      │
│  开始同居日期       │  IMM 5532, 租赁协议, 账单        │
│  分居期间(如有)     │  IMM 5532, 解释信                │
└─────────────────────┴──────────────────────────────────┘
```

### 1.3 地址信息一致性

| 检查项 | 涉及文件 |
|--------|---------|
| 担保人当前地址 | IMM 1344, 雇佣信, 银行账单, 驾照 |
| 申请人当前地址 | IMM 0008, 护照签证, 公用账单 |
| 共同居住地址 | 租赁协议, 账单, 政府文件, 照片元数据 |
| 历史地址 | IMM 5669, 前地址证明文件 |

## 二、表格间交叉检查

### 2.1 IMM 1344 与 IMM 5532 一致性

| IMM 1344 字段 | IMM 5532 对应 | 检查重点 |
|---------------|--------------|---------|
| 担保人婚姻状况 | Part A 关系历史 | 前婚数量一致 |
| 担保人子女 | Part A 家庭信息 | 子女数量/年龄一致 |
| 申请人信息 | Part B 申请人信息 | 基本信息完全一致 |
| 关系开始日期 | Part C 时间线 | 日期精确匹配 |

### 2.2 IMM 0008 与支持文件一致性

| IMM 0008 字段 | 支持文件 | 检查重点 |
|---------------|---------|---------|
| 出生信息 | 出生证明 | 完全一致 |
| 婚姻状况 | 结婚证/离婚证 | 状态和日期匹配 |
| 子女信息 | 子女出生证 | 数量和信息匹配 |
| 教育背景 | 学历证明 | 机构名称/日期匹配 |
| 工作经历 | 雇佣证明 | 公司/职位/日期匹配 |

### 2.3 IMM 5532 与陈述材料一致性

| IMM 5532 问题 | Love Story/陈述 | 检查重点 |
|---------------|----------------|---------|
| 如何认识 | 相识经过描述 | 方式/地点/介绍人一致 |
| 交往经过 | 发展历程描述 | 关键事件/日期一致 |
| 求婚/结婚 | 婚礼描述 | 地点/参与人/仪式一致 |
| 共同生活 | 日常生活描述 | 分工/习惯/安排一致 |
| 分居原因 | 分居解释 | 原因/时间/联系方式一致 |

## 三、支持文件间交叉检查

### 3.1 财务文件一致性

| 检查组合 | 验证点 |
|----------|--------|
| 银行账单 vs 雇佣信 | 工资金额匹配 |
| 银行账单 vs NOA | 总收入合理性 |
| 租赁协议 vs 银行账单 | 租金支付记录 |
| 汇款记录 vs 收入 | 汇款能力合理性 |

### 3.2 通讯记录一致性

| 检查项 | 验证方法 |
|--------|---------|
| 聊天记录时间 vs 电话账单 | 沟通时间匹配 |
| 聊天内容 vs 旅行证明 | 讨论与实际行程匹配 |
| 聊天频率 vs 关系声称 | 与亲密程度匹配 |
| 使用语言 vs 申请表声明 | 沟通语言一致 |

### 3.3 照片与其他证据一致性

| 照片元素 | 交叉验证 |
|----------|---------|
| 照片日期 | vs 机票、签证章、事件记录 |
| 照片地点 | vs 旅行记录、住址证明 |
| 照片人物 | vs 亲友信描述、社交媒体 |
| 照片事件 | vs IMM 5532 和陈述材料 |

## 四、常见矛盾类型与处理

### 4.1 轻微不一致（可接受，需解释）

| 类型 | 示例 | 处理建议 |
|------|------|---------|
| 日期微小差异 | 月份记忆误差（±1个月） | 提供解释 |
| 地址详略不一 | 省略城市 vs 完整地址 | 统一格式 |
| 姓名拼写变体 | 不同音译系统 | 提供对照说明 |
| 数字格式差异 | 日期格式（DD/MM vs MM/DD） | 明确说明 |

### 4.2 中等不一致（需要解释）

| 类型 | 示例 | 处理建议 |
|------|------|---------|
| 时间线空白 | 某段时间无证据覆盖 | 详细解释 + 替代证据 |
| 工作经历差异 | 公司名称不完全一致 | 提供公司更名证明 |
| 家庭信息差异 | 兄弟姐妹数量不一致 | 确认并更正 |

### 4.3 严重不一致（红旗）

| 类型 | 示例 | 影响 |
|------|------|------|
| 关键日期冲突 | 结婚日期与结婚证不符 | 可能导致拒签 |
| 身份信息冲突 | 出生日期多处不同 | 真实性质疑 |
| 关系声称冲突 | 同居时间与租约矛盾 | 可信度严重受损 |
| 前婚史隐瞒 | 离婚记录与声称不符 | 虚假陈述风险 |

## 五、自动检查清单

### 5.1 身份信息检查

```
□ 姓名拼写：护照 = 出生证 = 结婚证 = IMM 0008
□ 出生日期：护照 = 出生证 = IMM 0008
□ 国籍：护照 = IMM 0008 = IMM 5669
□ 婚姻状况：IMM 0008 = IMM 5532 = 结婚证/离婚证
```

### 5.2 时间线检查

```
□ 相识日期：IMM 5532 = Love Story = 支持信
□ 结婚日期：结婚证 = IMM 5532 = IMM 0008
□ 同居日期：IMM 5532 = 租赁协议 = 账单开始日期
□ 分居期间：IMM 5532 = 解释信 = 旅行记录
```

### 5.3 地址检查

```
□ 担保人地址：IMM 1344 = 雇佣信 = 驾照 = 银行账单
□ 申请人地址：IMM 0008 = 当地证明文件
□ 共同地址：租约 = 账单 = 政府文件
```

### 5.4 关系细节检查

```
□ 如何认识：IMM 5532 Part C #1 = Love Story
□ 求婚经过：IMM 5532 = Love Story = 照片说明
□ 婚礼描述：IMM 5532 = Love Story = 照片 = 亲友信
□ 沟通方式：IMM 5532 = 聊天记录 = 电话账单
```
