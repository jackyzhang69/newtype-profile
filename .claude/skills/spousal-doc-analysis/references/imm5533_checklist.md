# IMM 5533 文件清单验证规则

> 基于 IMM 5533 (09-2022) E - DOCUMENT CHECKLIST SPOUSE (INCLUDING DEPENDENT CHILDREN)

---

## Part A: 必需表格 (Forms Required)

### 担保人和申请人共同表格

| 表格 | 表格号 | 签名要求 | 验证点 |
|------|--------|---------|--------|
| Document Checklist | IMM 5533 | - | 必须上传 |
| Application to Sponsor | IMM 1344 | 担保人 + 申请人 | 原件签名，数字签名 |
| Application Fees | Receipt | - | 单一收据包含所有费用 |
| Financial Evaluation | IMM 1283 | 担保人 | 仅当配偶有自己有子女的子女时 |
| Generic Application Form | IMM 0008 | 申请人 | 数字表格 |
| Additional Family Info | IMM 5406 | 申请人 + 18岁以上家属 | 数字表格 |
| Schedule A | IMM 5669 | 申请人 + 18岁以上家属 | 数字表格 |
| Relationship Info | IMM 5532 | 担保人 + 申请人（多处签名） | 见下方详细签名位置 |
| Use of Representative | IMM 5476 | 指定代理人时 | 可选 |
| Authority to Release | IMM 5475 | 指定信息接收人时 | 可选 |

### IMM 5532 签名位置验证

```
担保人签名位置:
- Part A, #9 (Declaration of sponsor)
- Part C, #12 (Declaration of sponsor)

申请人签名位置:
- Part B, #5 (Declaration of applicant)
- Part C, #13 (Declaration of applicant)
```

## Part B: 支持文件 (Supporting Documents)

### B.1 担保人身份证明

#### 身份状态验证

| 身份 | 可接受文件 | 验证规则 |
|------|-----------|---------|
| 永久居民 | PR卡（正反面）或 CoPR/eCoPR | PR卡过期仍可接受 |
| 公民 | 公民证/卡、1977年前无照片公民卡、出生证明、护照 | 仅需一种 |
| 印第安人 | 印第安身份卡 | 或选择提供公民证明 |

#### 海外公民担保人

- [ ] 如担保人是海外公民，需提供证明将与配偶在加拿大居住的文件

### B.2 担保人前婚史

| 情况 | 必需文件 |
|------|---------|
| 离婚 | 最终离婚证书 |
| 婚姻无效 | 最终无效证书 |
| 前配偶去世 | 死亡证明 |
| 法定分居 | 分居证明（税表、法律文件） |

### B.3 担保人财务证明（非魁省居民）

**必需文件组合**：

- [ ] 雇佣信（说明雇佣期、工资、工时）或自雇财务文件
- [ ] 最近 Notice of Assessment (Line 15000) 或 Option C

**无法提供 NOA/Option C 时**：
- [ ] 书面解释 + 其他财务支持文件

### B.4 申请人身份文件

| 文件类型 | 要求 | 适用人员 |
|----------|------|---------|
| 护照/旅行证件 | 有效期页、照片页、签名页、加拿大入境章 | 申请人 + 随行家属 |
| 居留许可 | 如居住在非国籍国 | 申请人 |
| 出生证明/洗礼证明 | - | 申请人 + 所有子女 |
| 国民身份证 | 根据国家要求 | 申请人 + 家属 |
| 家庭户口本 | 根据国家要求 | 申请人 |

### B.5 申请人婚姻状态文件

| 当前关系 | 必需文件 |
|----------|---------|
| 法定婚姻 | 结婚证（政府注册，非仅宗教仪式证明） |
| 民事结合 | 民事结合证书 |

| 前段关系 | 必需文件 |
|----------|---------|
| 离婚 | 最终离婚证书/登记 |
| 婚姻无效 | 无效证书 |
| 前配偶/伴侣去世 | 死亡证明 |
| 曾被担保 | 书面说明（前担保人姓名、出生日期） |

### B.6 子女相关文件

| 情况 | 必需文件 |
|------|---------|
| 18岁以下随行子女（担保人非亲生父母） | IMM 5604 + 非随行父母身份证件 + 监护协议 |
| 22岁以上受抚养子女 | 依赖证明（身体/精神状况） |
| 收养子女 | 收养文件 + 习惯法收养仪式证据（如适用） |

### B.7 无犯罪记录证明

**适用人员**：申请人 + 18岁以上家属（非加拿大公民/PR）

**提供要求**：
- 18岁后在任何国家（加拿大除外）连续居住6个月以上
- 加拿大无犯罪证明由 IRCC 代为申请

### B.8 关系证明文件

#### 共同居住情况

**选择至少2类**：

- [ ] 共同房产证明
- [ ] 共同租赁协议
- [ ] 共同公用事业账户（水电、网络、电话）
- [ ] 共同银行/信用卡账户
- [ ] 共同车辆保险
- [ ] 政府文件显示同一地址（驾照等）
- [ ] 其他同地址文件（话费单、工资单、税表等）

**无法提供至少2类时**：
- [ ] 书面解释 + 其他同居证据

#### 非共同居住情况

必须提供：
- [ ] 联系证明（信件、短信、邮件、社交媒体，最多10页）
- [ ] 探访证明（机票、登机牌、护照出入境章）

#### 所有情况均需

- [ ] 照片（最多20张，不同时间地点，附说明）
- [ ] 至少2类以下文件：
  - 配偶认可文件（雇主保险福利等）
  - 财务支持/共同开销证据
  - 亲友认可证据（支持信、社交媒体等）
  - 过去同居证明（如目前未同居）

## 验证规则

### 语言要求

- [ ] 非英法文件必须附公证翻译
- [ ] 翻译需由认证翻译员提供或附宣誓书

### 文件格式要求

- [ ] 在线申请上传 PDF 格式原件
- [ ] 不提交原件（除非特别要求）

### 国家特定要求

- [ ] 检查 IRCC 国家特定要求页面
- [ ] 根据居住国和国籍国要求提供额外文件

## 简化要求条件

如果**同时满足**以下4个条件，可不提供额外关系证明：

1. [a] 目前与担保人共同居住 ✓
2. [b] 与担保人有共同子女 ✓
3. [c] 双方均为首次婚姻 ✓
4. [d] 申请日婚姻已满2年 ✓

**满足全部4条件**：仅需基本文件，无需额外照片和关系证明

**不满足全部4条件**：需提供完整关系证明材料
